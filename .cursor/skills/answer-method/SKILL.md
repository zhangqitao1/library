# 能力 / 答题方法专项

> 知识点会背 ≠ 会考试。本技能补"**方法与能力**"这一层：阅读、完形、作文、题型套路等**可迁移的答题框架**。
> 方法直接对接命题新方向（见 `AGENTS.md` 九.4）——新题"长情境+高阶+跨学科+要过程"，最常丢分的两处就是"读不懂材料"和"答得不规范"。
> 现成方法文档放在 `subjects/<科目>/能力方法/`（单科），**跨科通用方法**放 `subjects/通用能力方法/`（册次无关，公开通用、无个人信息）。已有：
> - 🔑 通用（最优先，几乎每科都用）：`通用能力方法/信息提取四步法.md`（读真实长情境/非连续文本）、`通用能力方法/主观题过程规范.md`（拿过程分）
> - 语文 `现代文阅读答题框架.md`、`作文结构与升格.md`
> - 英语 `完形与阅读解题策略.md`
> 还没有的能力点 → 按本技能现场教，并建议补一份方法文档。

## 🔴 两个新形势核心能力（跨科，遇到长材料/主观题优先带）
1. **信息提取**（对应 9.4②）：题给真实长材料先别慌，按 `通用能力方法/信息提取四步法.md`——带着问题读→圈关键信息→设问对应材料→整合作答。
2. **过程规范**（对应 9.4④）：主观题"大意对却扣分"，按 `通用能力方法/主观题过程规范.md`——分点+术语+有依据，并套各科结构模板（历史论从史出、道法观点+理由+结论、地理先因后果、数学写步骤等）。

## 用法（引导式，别直接灌框架）

1. **定能力点**：她哪类题反复丢分？（阅读概括题、赏析题、完形、作文结构、某数学题型…）先读该科 `能力方法/` 对应文档 + `复习流程.md`。
2. **先暴露她的做法**：拿一道真题，先让她**按自己的方法做一遍**，说出"我是怎么想的"。**先诊断再教**——她的卡点决定教什么。
3. **再给框架（脚手架式）**：对照方法文档的步骤，一步一步带，不一次倒完。每步问"这步你来试"，让她自己填。
4. **当场套用 2~3 道变式**：同一框架换不同材料/题目练，直到她能独立走完整套步骤（能迁移才算会）。
5. **她讲一遍框架**（费曼）：让她当小老师说"这类题分几步、每步干嘛"。

## 命题/出练习
- 要成套练习时转 `exam-maker`，难度配比见 `AGENTS.md` 九.2。
- 方法专项练习重"同框架、多材料"，不堆同一篇。

## 闭环
- 用错框架/仍丢分 → `mistake-log` 入错题本，错因多归"方法"类。
- 更新 `<当前册>/00-册地图.md` 与学情薄弱点；`学情/复习日志.md` 追加一行。

## 新增方法文档时
- 存 `subjects/<科目>/能力方法/<名称>.md`，**只写通用答题方法/思维步骤**（不含个人信息，遵守 `AGENTS.md` 八公开边界）。
- 方法是通用技能、可公开；涉及具体课文/真题事实时拿不准标 `⚠️待核对`，不编造。
