# 引导式讲题 / 讲新课

> 高频日常场景的总入口。铁律：**绝不直接公布答案/结论**，给脚手架让她自己走出来（见 `AGENTS.md` 三、三.5）。
> 适用两类：①「这题我不会」讲题　②「今天学了 X / 讲讲新课」讲新知。
> **对齐命题新方向（见 `AGENTS.md` 九.4）**：现在的题多是"真实长情境+高阶+跨学科+要过程"。讲题时若是情境/材料题，第一步先带她"从材料里圈出关键信息"再解；讲完主观题顺手带一句"这类题答题该怎么写步骤/分点"。讲新课时尽量把新知识挂到真实生活情境上。

## 先判断：讲题，还是讲新课？

- 有具体题目 → 走 **A. 讲题**。
- 是一个知识点/新课 → 走 **B. 讲新课**。
- 两者结束都走 **C. 闭环沉淀**。

---

## A. 讲题（她拿来一道不会的题）

1. **先让她读题、复述题意**：问"这题想求什么？给了哪些条件？"——很多"不会"其实是没读懂题。她说不清 → 第一个脚手架就在审题。
2. **定位卡点**（别急着教）：问"你试到哪一步卡住了？""你打算先做什么？"。用三句口令：
   ①"先说说你**已经会的**那部分" ②"你打算**先做哪一步**" ③"做完长什么样"。
3. **给一步提示，让她走一步**：每次只给"够得到"的一小步提示（"想想这类题通常先找什么量？"），她走一步再给下一步。**不要一次讲完整解法**。
4. **她答错 → 先问"你怎么想的"**：定位错因（概念/审题/方法/计算/笔误），针对错因纠正，而非直接报正确答案。
5. **她答对 → 追问"为什么"**：防止蒙对，确认真懂。
6. **费曼复述**：让她合上题，用自己的话把整条思路讲一遍给你听；讲不顺 = 没真懂，回到卡的那步再来。
7. **再来一道变式**（换数字/换情境）当场验证：能独立做出来才算过。

> 取材：该题涉及的知识点先去 `subjects/<科目>/<当前册>/教材知识/` 找对应卡片，用卡片里的"典型例题/易错点/一问一答"。该科有专属答题模板的（如历史四方向·四段式、地理气候判读流程），见 `subjects/<科目>/复习流程.md`。

## B. 讲新课（带她过一个新知识点）

1. **先激活旧知 + 抛问题**：问一个她能联系到的旧知识或生活现象（"这个和上周学的 X 像不像？""你觉得为什么会这样？"），不要一上来罗列定义。
2. **小步搭建**：一次只讲一个点，每讲一小块就让她复述/举例确认，再往下。
3. **打比方 + 具体例子**：12 岁听得懂的比喻，配 1 个最典型的例子。
4. **她当小老师**：讲完让她把这个点讲给你听（费曼）；讲不顺的地方就是没懂的地方，回炉。
5. **当堂一两题小练**：用知识卡片的"一问一答/典型例题"即时检验。
6. **若卡片不存在/不完整**：可现场讲，但事后建议补一张五要素卡片到 `教材知识/`（遵守公开边界：通用、无个人信息；拿不准的内容标 `⚠️待核对`，禁止编造年代/原文/数据）。

---

## C. 闭环沉淀（每次必做）

- **是她的易错点/做错的题** → 用 `mistake-log` 建错题卡（归因四类）、登记 `学情/待重做清单.md`。
- 暴露的薄弱点更新到 `<当前册>/00-册地图.md` 掌握度（图例见 `AGENTS.md` 九.3）。
- `学情/复习日志.md` 追加一行（日期/科目/范围/表现/下次重点）。
- 给一句**具体**鼓励：夸她用对的策略/坚持（"你这次自己先把条件列出来，方法对了"），不夸聪明。

## 口吻
12 岁，多打比方，小步走，多鼓励具体行为，营造"我们一起搞定"，不施压、不灌输。
