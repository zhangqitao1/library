# 生成试卷

## 步骤

### 1. 定范围与目标
- 普通自测：覆盖指定单元的主要知识点。
- **针对性自测（优先）**：读 `<当前册>/00-册地图.md` 与 `<当前册>/错题本/`，专攻 ❌/🔶 的薄弱点——这种卷子提分效率最高。（当前册见学情档案）

### 2. 命题原则（对标济南市质量监测命题导向）
- **难度配比见 `AGENTS.md` 九.2**（分数科 6:3:1 / 等级科 7:2:1），此处不再重复，按当前科目套用。
- **🔴 命题方向见 `AGENTS.md` 九.4**（2026 素养立意：①减记忆增高阶 ②长情境先提取 ③跨学科+动态/探究 ④重过程规范）——这是出卷的方向总纲，每份卷都要显著体现这四条，**不要只仿往年真题套路**，同一考点换情境出。出题专属落点：
  - 高阶题用"为什么/比较/怎么用"设问；纯识记题压到最低。
  - 给真实长材料，先设"信息提取"小问再设解题小问。
  - 数学配动态几何（旋转/折叠/隐圆）/新定义/三级递进探究；理科配实验探究（原理/数据/误差/改进）。
  - 主观题在题干或解析里**显式要求并示范**过程规范（分点、术语、论从史出/观点+理由+结论/写步骤）。
- **每题标注考查知识点**，方便做完反查薄弱点。
- 题目来源优先：① 错题的变式（换数字/换情境，测是否真会）② 教材知识卡片里的"变式·拓展"③ 典型题。
- 题量建议：小自测 8~12 题；单元卷 20~30 题。
- 答案配**简要解析**，解析体现思路而非只给结果。

### 3. 输出 HTML（统一格式，便于周期性测试）
- 复制 `templates/试卷模板.html`（已是"验证过的优质骨架"：复习资料+自测两段、吸顶按钮、打印CSS、题块样例），替换占位、按题块样例填题。**别另起炉灶简化它**——稳定优质靠复用这套骨架。
- 必须**自包含**（样式内联、不依赖网络）、可打印、答案默认折叠（点击展开）。
- **🔴 公式只用 Unicode 上下标**（`a<sup>2</sup>`、x²、½、≥、∥、≌、°、÷），**禁止 LaTeX / `\( \)` / MathJax**——HTML 离线打开没有公式库，LaTeX 会显示成乱码。这是已踩过的坑。
- **🔴 无法插图的题**（几何图、坐标/图象、地图）→ 用文字把图的关键信息讲清（各点/各段坐标、起止值、方位、数据），**不要写"如图"却没有图**。
- **每题必带** `<span class="kp">[考点：…]</span>`。
- **卷头 meta 必须含**：科目 · 范围 · **日期** · **测试类型** · 建议限时 · 难度配比。
- 存到 `subjects/<科目>/<当前册>/试卷/自测卷/`。
- **命名规范**：`YYYY-MM-DD_范围_类型.html`
  - 日期＝出卷日（系统当天）；范围＝考查内容（如 `第一单元`/`期中1-3单元`/`错题重做`）；
  - 类型 ∈ {`自测`,`周测`,`单元卷`,`月测`,`期中`,`期末`,`单元专项`,`错题重做`}。
  - 例：`2026-09-22_错题重做_第2次.html`、`2026-10-10_月测_1-3单元.html`。

### 4. 登记到试卷清单（必做，支撑周期性测试）
- 在 `subjects/<科目>/<当前册>/试卷/index.md`（无则按下表头新建）**最上面一行**追加本次记录：

  ```
  | 日期 | 范围 | 类型 | 链接 |
  |---|---|---|---|
  | YYYY-MM-DD | 范围 | 类型 | [打开试卷](自测卷/文件名.html) |
  ```
- 该 `index.md` 即网站上"试卷"栏目页，历次测试一目了然，方便按 1→3→7→15 天或单元/月度节奏回测。

### 4.5 出卷后自检清单（🔴 交付前逐条核对，专治"偶发残片/答非所问"）
1. **残片扫描**（在仓库根跑，应无输出）：
```
f="subjects/<科目>/<册>/试卷/自测卷/<文件名>.html"
grep -nE '[0-9°a-zA-Z²³]+[+\-−=][.]{2,3}|\|\||＝＝|class="(ans|sol)">\s*</div>|如图(（[^）]*）)?[^，。、：]{0,4}。' "$f"
```
　查的是：算式中途断成 `…=5+...` 的残片、误入的双竖线、空答案/空解析、写了"如图"却无图。
2. **逐题对答**：随机抽 3~5 题，确认答案确实回答了题干、解析步骤能推出该答案（别答非所问）。
3. **诚信**：猜题/预测题已标"非真题"；存疑知识点带 `⚠️待核对`，无编造的年代/原文/数据。
4. **配比与考点**：题量与难度配比符合学科（分数科6:3:1／等级科7:2:1）；每题都有 `[考点：…]`。
5. **能力可选**：用浏览器起本地服务（`python -m http.server`）打开，点"显示全部答案"确认 16/16 能展开、排版正常。
> 任一条不过 → 当场改，别把带瑕疵的卷子交给小雨（她对答案时会被误导）。

### 5. 交付与结果沉淀
- 告诉小雨：先独立做，做完点"显示答案"对照。
- **她的真实分数/结果记到本地**：错题进对应 `<当前册>/错题本/`，并向 `学情/复习日志.md` 追加一行（日期/科目/范围/表现/下次重点）。**这些是本地专属，不上传公网。**
- 做错的题让她反馈回来，或拍照走 `paper-scan`。

## 提示
- 一份针对性小卷 + 隔几天的重做，比一张大综合卷有用得多。
- 若某知识点她已连续做对，可减少该点题量，把空间留给薄弱点。
