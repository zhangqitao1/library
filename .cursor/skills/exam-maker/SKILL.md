# 生成试卷

## 步骤

### 1. 定范围与目标
- 普通自测：覆盖指定单元的主要知识点。
- **针对性自测（优先）**：读 `<当前册>/00-册地图.md` 与 `<当前册>/错题本/`，专攻 ❌/🔶 的薄弱点——这种卷子提分效率最高。（当前册见学情档案）

### 2. 命题原则（对标济南市质量监测命题导向）
- 难度配比（对标济南难度系数比）：
  - **分数学科**（语数英物化）：基础 60% / 中档 30% / 压轴 10%（= 6:3:1）。
  - **等级学科**（生物/地理/史/道法）：基础 70% / 中档 20% / 压轴 10%（= 7:2:1，更重基础）。
- **情境化、反套路**（济南/历下命题趋势）：多设真实生活/跨学科情境，考"理解+迁移"而非死记；**不要只仿往年真题套路**，同一知识点换情境出，专治"会了这题、换个说法就不会"。
- **每题标注考查知识点**，方便做完反查薄弱点。
- 题目来源优先：① 错题的变式（换数字/换情境，测是否真会）② 教材知识卡片里的"变式·拓展"③ 典型题。
- 题量建议：小自测 8~12 题；单元卷 20~30 题。
- 答案配**简要解析**，解析体现思路而非只给结果。

### 3. 输出 HTML
- 复制 `templates/试卷模板.html`，替换占位，填入题目/答案/解析。
- 必须**自包含**（样式内联、不依赖网络）、可打印、答案默认折叠（点击展开）。
- 存到 `subjects/<科目>/<当前册>/试卷/自测卷/`，命名：`YYYY-MM-DD_范围_针对X.html`。

### 4. 交付
告诉小雨：卷子已生成，先独立做，做完点"显示答案"对照；做错的会自动进错题本（让她做完把错题反馈回来，或拍照走 `paper-scan` 识别卷子）。

## 提示
- 一份针对性小卷 + 隔几天的重做，比一张大综合卷有用得多。
- 若某知识点她已连续做对，可减少该点题量，把空间留给薄弱点。
