# 思维导图

## 步骤

### 1. 取材
读 `subjects/<科目>/<当前册>/教材知识/` 对应单元的知识卡片，提炼"主干 → 分支 → 要点"三层结构。（当前册见学情档案）

### 2. 生成大纲
按 `templates/思维导图模板.md` 写成 markdown 大纲（# 单元 / ## 主干 / ### 分支 / - 要点），存到 `subjects/<科目>/<当前册>/思维导图/<单元>.md`。

### 3. 可视化
告诉用户：把大纲贴到 https://markmap.js.org 或支持 markmap 的工具即可渲染成图；也可导入 XMind。

### 4. 用导图做主动回忆（关键）
不要只给图。用导图带小雨做"主动回忆"：
- 遮住分支，只给主干，让她说出下面有哪些分支。
- 说不全的地方，正是知识网络的漏洞 → 标记为薄弱点更新到学情。

## 提示
- 导图是"检验知识是否成网"的工具，不是装饰。重点在让她自己复述结构。
