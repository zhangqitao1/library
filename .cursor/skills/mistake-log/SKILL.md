# 错题归集

错题不是抄一遍，而是要**闭环清零**。

## 步骤

### 1. 建错题卡
按 `templates/错题卡模板.md` 在 `subjects/<科目>/<当前册>/错题本/` 建卡（当前册见学情档案，现为"七年级上册"），命名：`<单元>_<一句话摘要>.md`。

### 2. 归因（最重要）
判断错因属于哪类，写进 frontmatter `错因`：
- **概念不清**：定义/原理没懂 → 回知识卡片重学。
- **审题**：看错/漏条件 → 练读题划重点。
- **方法**：思路方向错 → 补该题型套路。
- **计算**：方向对算错 → 练计算/检查。
- **笔误**：会但写错 → 提醒规范。
> 错因比对错本身更重要——它决定怎么补。

### 3. 关联知识点
`知识点链接` 指向对应知识卡片，形成"错题 ↔ 知识点"双向网。

### 4. 安排重做（间隔复习）
- 状态设 `未掌握`，按 1天-3天-7天-15天 提示重做。
- 重做用**变式**（换数字/情境），防止背答案。
- 每次重做把结果追加进 `重做记录`。

### 5. 清零规则
连续 2 次重做全对 → 状态改 `已清零`。

### 6. 联动
- 更新 `<当前册>/00-册地图.md` 掌握度与"薄弱点 TOP"。
- 月底把所有"未清零"错题汇总，用 `exam-maker`（生成试卷）技能出一张专项清零卷。

## 提示
同一知识点反复出错 → 这是"假性掌握"，要回到知识卡片用引导式重讲，而不是反复刷同类题。
