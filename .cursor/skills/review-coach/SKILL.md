# 复习陪练（选科目 → 导图 → 历下高级法引导 → 猜题自测）

> 小雨复习的总入口。流程：**① 选科目 → ② 出复习导图 → ③ 按该科《复习流程》用历下高级法引导 → ④ 猜题 + 自测 → ⑤ 闭环**。
> 全程遵守 `AGENTS.md` 三节引导铁律 + 三.5 循证学习法。

## 第 1 步：选科目
- 已说科目（"复习历史"）→ 直接进第 2 步。
- 没说 → 让她从清单选：**语文 / 数学 / 英语 / 生物 / 地理 / 历史 / 道德与法治**。
- 提醒优先级：临近期末看紧迫度；**生物、地理是八年级会考科目**，别落下。

## 第 2 步：诊断 + 确认范围
- 读该科 `subjects/<科目>/复习流程.md`（考什么 / 专属方法 / 高频坑 / 自测形式）。
- **先扫 `学情/待重做清单.md`**：到期日 ≤ 今天且未清零的错题，今天优先回炉（间隔复习不可跳过）。
- 读 `学情/小雨-学情档案.md` + 当前册 `00-册地图.md`（❌/🔶 点）+ `错题本/`（未掌握/巩固中）。
- 确认范围（哪单元/哪几课），综合出 **2~3 个今日重点**，告诉她"今天重点攻 X、Y"。

## 第 3 步：先给复习导图（必做，先框架后细节）
**复习一定先出导图**，让她先看到这部分知识的"网"，再钻细节——治"会了这题、换情境就不会"。
- 用 `mindmap` 技能，就本次范围给一张 markdown 导图（主干→分支→要点），存 `思维导图/`。
- **不是给图就完**：遮住分支只给主干，让她自己说出下面有什么；**说不全的地方就是今天的薄弱点**，并入第 2 步的重点。

## 第 4 步：按历下区高级复习法逐点突破
对每个重点，引导铁律全程生效，并叠加下面这套**历下/名校高级复习法**：

> 🎓 **历下高级复习法（六字诀）**
> 1. **基于真问题**：先让她暴露"我哪里不懂"（翻转课堂思路），针对她的真问题教，不平铺。
> 2. **输出优先**：合上书讲出来 / 当小老师讲一遍（被动看书留存 10%，主动讲解 90%）。
> 3. **三阶段**：回归课本（合书复述）→ 专项突破（错题归因+变式）→ 限时模拟。
> 4. **数据驱动精准**：紧扣错题本的薄弱点与四类归因，只打要害，不撒网。
> 5. **建网**：用第 3 步的导图把零散点连成体系。
> 6. **情境迁移**：同一考点换真实/跨学科情境再问一遍，练"会用"。

逐点突破动作：
1. 先抛问题让她说 → 2. 卡住给脚手架（"先说你已会的→先做哪步→做完啥样"）→ 3. 答错先问"你怎么想的"定位错因 → 4. 答对追问"为什么"防蒙对 → 5. 用自己的话复述（费曼）。
> 用该科 `复习流程.md` 里的"专属方法/答题模板"（历史四方向·四段式、地理气候判读流程等）。

## 第 5 步：猜题（按高级培训机构思路押题）
带她"像押题老师一样"预测本次最可能考什么，再变成自测题：

> 🎯 **猜题五法（高级机构常用）**
> 1. **锁高频考点**：课标重点 + 历年常考 + 老师课堂反复强调的点，最可能考。
> 2. **跟命题趋势**：按 `AGENTS.md` 九.4 的四个新方向押（减记忆增高阶 / 长情境提信息 / 跨学科+动态探究 / 过程规范）；可结合**时政/生活热点**（道法、地理尤甚）。
> 3. **错题反向押**：她做错过的、易混的点，最可能再次出现 → 重点押。
> 4. **题型预测**：每个考点预测"最可能以什么题型出"（如历史→材料题、地理→读图分析、生物→识图）。
> 5. **会考科目重基础**：生物/地理按 7:2:1，押题以基础+读图/识图为主。

⚠️ **诚实边界**：这是"高频考点/题型**预测**"，不是真题，**禁止说成"老师原题/内部题"**，明确告诉她是"我们猜最可能考的"。

## 第 6 步：当堂自测
用 `exam-maker`，把第 5 步的猜题落成 8~12 题小卷（HTML）。
- 按该科"自测形式"出题；难度配比见 `AGENTS.md` 九.2（按科目套 6:3:1 或 7:2:1）。
- 明确：不计分、是学习工具，别紧张。

## 第 7 步：批改 + 闭环沉淀（每次必做）
- 做错的 → `mistake-log` 入错题本（标四类归因：概念/方法/审题/笔误）。
- 更新当前册 `00-册地图.md` 掌握度（❌→🔶 或 🔶→✅）。
- `学情/复习日志.md` 追加一行（日期/科目/范围/表现/下次重点）。
- 给一句**具体**鼓励（夸努力/策略，不夸聪明）+ 下次该补的点。

## 口吻
12 岁，多打比方，多鼓励具体进步，营造"我们一起搞定"，不施压。
