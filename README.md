# 小雨的学习知识库

一个给 12 岁孩子（济南·六三制七年级）用的**私人 AI 家教知识库**。
本仓库**不含任何程序代码**，全部是 markdown 规则与知识——用 Cursor / Claude Code 等 AI agent 打开它，agent 读了规则就会"教"。

## 怎么用（家长视角，零负担）

1. 用 Cursor 或 Claude Code 打开本文件夹。
2. 直接对 AI 说人话，比如：
   - 「我明天考历史，帮我复习」
   - 「这是我今天的数学卷子」（拖入照片）
   - 「这道题我不会」
   - 「帮我把这周的错题整理一下」
3. AI 会自动读 `AGENTS.md` 里的家教规则，按"先诊断→引导→闭环沉淀"的方式陪练。

> 核心：你只管把卷子拍照丢进来、让孩子开口问。识别、归类、错因分析、出针对性卷子、记录进步，AI 全包。

## 目录结构

```
library/
├── AGENTS.md            ← 家教系统总规则（AI 必读，最重要）
├── .cursor/
│   ├── rules/           ← Cursor 自动加载的规则
│   └── skills/          ← 技能（复习陪练/识卷/出卷/导图/错题，Cursor 自动发现）
├── 学情/                ← 小雨的学情档案 + 复习日志（AI 的"记忆"）
├── templates/           ← 知识卡片 / 错题卡 / 学科地图 / 试卷HTML 模板
└── subjects/            ← 各科知识库（七上已全部建成）
    ├── 历史/ 语文/ 数学/ 英语/ 生物/ 地理/ 道德与法治/
    └── _未来科目/        ← 物理(初二)、化学(初三)占位
```

## 每科内部结构（按学期分册，随孩子逐年级生长）

```
subjects/<科目>/
├── 00-学科总览.md          ← 跨年级导航（七上/七下/八上…各册进度）
└── 七年级上册/             ← 一个学期一个文件夹；升学期=新建一个
    ├── 00-册地图.md        ← 本册知识树 + 掌握度看板（一眼看薄弱区）
    ├── 教材知识/<单元>/     ← 按单元/课的知识卡片（五要素）
    ├── 思维导图/
    ├── 易错点/
    ├── 错题本/             ← 收集→归因→重做→清零 闭环
    └── 试卷/
        ├── 原卷/           ← 拍照原件
        ├── 结构化/         ← 识别后可检索版
        └── 自测卷/         ← AI 生成的 HTML 卷子
```
