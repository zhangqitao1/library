# 小雨的学习知识库

一个给 12 岁孩子（济南·六三制七年级）用的**私人 AI 家教知识库**。
主体是 markdown 规则与知识——用 Cursor / Claude Code 等 AI agent 打开它，agent 读了 `AGENTS.md` 里的规则就会按"先诊断→引导→闭环沉淀"的方式**教**，而不是把答案倒出来。

> 仓库里也有少量"工程件"：把知识库一键发布成网站的 `mkdocs.yml` / `.github/workflows/deploy.yml`，以及自动记录提问的 `.cursor/hooks/`。家长无需关心这些，正常用 AI 对话即可。

## 怎么用（家长视角，零负担）

1. 用 Cursor 或 Claude Code 打开本文件夹。
2. 直接对 AI 说人话，比如：
   - 「我明天考历史，帮我复习」
   - 「这是我今天的数学卷子」（拖入照片）
   - 「这道题我不会」
   - 「帮我把这周的错题整理一下」
   - 「小雨这周都问了些什么」（看提问周报）
3. AI 会自动读 `AGENTS.md` 的家教规则，按引导式教学陪练，并在结束时把进度沉淀到 `学情/`。

> 核心：你只管把卷子拍照丢进来、让孩子开口问。识别、归类、错因分析、出针对性卷子、记录进步，AI 全包。

## 🔒 公开 vs 本地（重要边界）

本库可发布成网站方便随时查阅，因此把"教学资源"和"孩子隐私数据"严格分开：

| 内容 | 位置 | 是否会上网 |
|---|---|---|
| 教材知识卡片 / 思维导图 / 易错点 / 自测卷 | `subjects/` | ✅ 会发布（通用资源，不含个人信息） |
| 学情档案 / 复习日志 / 待重做清单 / 提问记录 | `学情/` | ❌ 仅本地（已 `.gitignore`） |
| 孩子真实错题与重做记录 | 各册 `错题本/` | ❌ 仅本地（已 `.gitignore`，公开库仅用模板示范格式） |

> 详细边界铁律见 `AGENTS.md` 第八节。AI 绝不把孩子姓名/真实错题/学情写进 `subjects/`。

## 网站发布（可选）

- 推到 `main` 分支后，`.github/workflows/deploy.yml` 会用 MkDocs 自动构建并发布到 GitHub Pages。
- 只发布 `subjects/`（`mkdocs.yml` 里 `docs_dir: subjects`），`学情/`、`错题本/` 不在其中，**不会上网**。
- 本地预览：`pip install -r requirements.txt` 后运行 `mkdocs serve`。

## 目录结构

```
library/
├── AGENTS.md            ← 家教系统总规则（AI 必读，最重要）
├── README.md            ← 本文件
├── .cursor/
│   ├── rules/           ← Cursor 自动加载的规则（含提问追踪）
│   ├── skills/          ← 技能（复习陪练/讲题/识卷/出卷/错题/导图/能力方法/家长周报，Cursor 自动发现）
│   └── hooks/           ← 提问自动记录钩子
├── 学情/                ← 小雨的学情档案+日志+错题（AI 的"记忆"，仅本地）
├── templates/           ← 知识卡片 / 错题卡 / 册地图 / 学科总览 / 试卷HTML 模板
├── subjects/            ← 各科知识库（会发布成网站）
│   ├── 历史/ 语文/ 数学/ 英语/ 生物/ 地理/ 道德与法治/
│   └── _未来科目/        ← 物理(初二)、化学(初三)占位
├── docs/                ← 阶段计划（建库任务、暑假衔接等）
├── mkdocs.yml           ← 网站配置
└── .github/workflows/   ← GitHub Pages 自动部署
```

## 每科内部结构（按学期分册，随孩子逐年级生长）

```
subjects/<科目>/
├── 00-学科总览.md          ← 跨年级导航（七上/七下/八上…各册进度）
├── 复习流程.md             ← 该科专属复习打法（册次无关）
├── 能力方法/               ← 答题框架/解题策略（阅读/完形/作文等，册次无关）
└── 七年级上册/             ← 一个学期一个文件夹；升学期=新建一个
    ├── 00-册地图.md        ← 本册知识树 + 掌握度看板（一眼看薄弱区）
    ├── 教材知识/<单元>/     ← 按单元/课的知识卡片（五要素）
    ├── 思维导图/
    ├── 易错点/
    ├── 错题本/             ← 收集→归因→重做→清零 闭环（真实数据仅本地）
    └── 试卷/
        ├── 原卷/           ← 拍照原件
        ├── 结构化/         ← 识别后可检索版
        └── 自测卷/         ← AI 生成的 HTML 卷子
```
