## 6. 验收清单（每批产出后必查）
1. **结构**：内容都在 `subjects/<科目>/七年级下册/` 下，未污染其他科目/册。
2. **断链检查**（在仓库根运行）：
```
while IFS= read -r map; do dir=$(dirname "$map");
  grep -oE '\]\(([^)]+\.md)\)' "$map" | sed -E 's/\]\(//; s/\)//' | while read -r l; do
    [ -f "$dir/$l" ] || echo "断链: $map -> $l"; done
done < <(find subjects -name '00-册地图.md'); echo done
```
　输出只应有 `done`，无"断链"。
3. **格式**：随机抽查 3~5 张卡片，五要素齐全、frontmatter 字段完整、`年级: 七年级下册`。
4. **诚信**：搜索 `⚠️待核对` 应只出现在确实存疑处；不得有编造的年代/原文/数据。
5. **总览同步**：各科 `00-学科总览.md` 的七下行已更新。
6. 更新 `README.md` 当前状态 与 `学情/复习日志.md` 追加一条建库记录。

- **册次路径**：所有新内容走 `subjects/<科目>/<册次>/...`，别退回扁平结构。
