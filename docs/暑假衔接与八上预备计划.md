# 暑假衔接与八上预备计划（七下期末后启用）

> 面向"七下期末考完 → 升八年级开学"这段暑假。目标不是提前刷完八上，而是**补七下漏洞 + 为八年级三件大事卡好位**：①物理开课 ②生物/地理会考前置（八下会考、学完即考、计入中考）③学习强度上台阶。
> 启用时机：七下期末出分后。届时把本计划落成具体周计划，并在 `学情/小雨-学情档案.md` 把"当前学期"改为"八年级上册"、新建册文件夹。

---

## 一、暑假三件事（按优先级）

### 1. 七下错题清零（最高优先，承上）
- 把 `学情/待重做清单.md` 里所有"未清零"错题做完两轮变式 → 全部清零。
- 用 `exam-maker` 出 1~2 张"七下薄弱点专项清零卷"，重点是期末暴露的 ❌/🔶 点。
- 同步回填各册 `00-册地图.md` 掌握度，让八上开学时学情档案是"干净"的。

### 2. 生物 / 地理——会考科目，暑假别断（重中之重）
- 六三制：**八下学期末生地会考，学完即考、计入中考**。七下这两科是会考范围的一部分，**暑假必须保温**。
- 动作：把七上+七下生物/地理过一遍思维导图（合书复述），易错点各扫一遍；不求新进度，求"不遗忘 + 读图/识图熟练"。
- 心态：明确告诉她这两科"现在抓 = 给中考存钱"，不是副科。

### 3. 物理预备（八上新开课，启蒙优先）
- 八上新增物理，是很多孩子的分水岭。暑假**轻启蒙**即可：从生活现象入手（声、光、热、运动），培养"观察—提问—解释"的兴趣，**不提前死磕公式**。
- 升学期时新建 `subjects/物理/八年级上册/`，按现有"五要素卡片 + 思维导图 + 易错点 + 册地图"标准建库（沿用历史七上质量标杆）。

## 二、升八年级的系统动作（开学前做）

1. 改 `学情/小雨-学情档案.md`：当前学期 → "八年级上册"；更新各科状态与薄弱点。
2. 新建各科 `八年级上册/` 文件夹（含 `00-册地图.md`），**七年级两册自动转为复习库，只读不改**。
3. 新建 `subjects/物理/`（学科总览 + 八上册）。
4. 联网核对八上各科 2024 版真实目录后再建卡（拿不准标 `⚠️待核对`，禁止编造）。

## 三、暑假节奏建议（防止假期"学崩"或"全荒"）
- 每天小份量、固定时段；**会考科（生地）+ 物理启蒙 + 七下错题**三者轮换，交错混合（提取网格）。
- 自测一律不计分、低压力；多用"当小老师讲一遍"做费曼检验。
- 注意护眼与休息，别一次拖太久。
