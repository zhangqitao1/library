# 历史 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**历史专属打法**。
> 历史 = 脉络 + 理解型，复习核心是"**串时间线 + 抓因果 + 史料会读**"，不是孤立背年代。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。
> 已知薄弱（演示）：丝绸之路路线、汉武帝大一统措施——综合复习时优先扫。

## 二、考什么（五大维度）
1. **时间线 / 朝代脉络**：朝代顺序、重大事件先后。
2. **政权并立表**：民族、时间、建立者、都城、灭亡（辽宋夏金元最爱考）。
3. **人物事迹配对**：谁做了什么。
4. **史料 / 材料解读**：从材料里提取信息。
5. **原因与影响分析**：背景—措施—作用—认识。

## 三、历史专属方法（怎么高效）
- **时间轴 + 表格串记**（最重要）：把零散事件挂到时间轴上；并立政权、改革措施用表格对照，建网。
- **人物事迹配对法**：先定朝代 → 再定民族/方向 → 最后定具体事件。
- **答题模板（理解后用）**：
  - 边疆治理题 → 按 **东南(台湾)—西南(西藏)—西北(新疆)—东北(中俄)** 四方向答。
  - 改革/制度题 → **背景—措施(分类)—作用—认识** 四段式。
- **输出学习法**：让她用自己的话讲一条线索（如"唐朝由盛转衰怎么发生的"）。
- **理解因果**：多问"为什么会……""带来什么影响"，不死记结论。

## 四、高频坑（优先攻）
- **朝代顺序**记乱。
- **相近事件/人物混**（如丝绸之路路线顺序、各政权对应民族）。
- **时间易混点**（郑成功收台 1662 ≠ 清设台湾府 1684）。
- 材料题脱离材料空答。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按单元过时间线+核心事件，合书复述一条线索。
2. **专项突破**：错题归因，专攻易混事件/政权表/配对题。
3. **限时模拟**：限时整卷，重点练材料分析题。

## 六、当堂自测形式（交给 `exam-maker`）
时间排序、政权并立填表、人物事件配对、材料解读、影响简答。

## 七、闭环沉淀
错题入 `错题本/`（易混事件、配错的人物单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
