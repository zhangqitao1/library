# 历史 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**历史专属打法**。
> 历史 = 脉络 + 理解型，复习核心是"**串时间线 + 抓因果 + 史料会读**"，不是孤立背年代。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。
> 已知薄弱（演示）：丝绸之路路线、汉武帝大一统措施——综合复习时优先扫。

## 二、考什么（五大维度）
1. **时间线 / 朝代脉络**：朝代顺序、重大事件先后。
2. **政权并立表**：民族、时间、建立者、都城、灭亡（辽宋夏金元最爱考）。
3. **人物事迹配对**：谁做了什么。
4. **史料 / 材料解读**：从材料里提取信息。
5. **原因与影响分析**：背景—措施—作用—认识。

## 三、历史专属方法（怎么高效）
- **时间轴 + 表格串记**（最重要）：把零散事件挂到时间轴上；并立政权、改革措施用表格对照，建网。
- **人物事迹配对法**：先定朝代 → 再定民族/方向 → 最后定具体事件。
- **答题模板（理解后用）**：
  - 边疆治理题 → 按 **东南(台湾)—西南(西藏)—西北(新疆)—东北(中俄)** 四方向答。
  - 改革/制度题 → **背景—措施(分类)—作用—认识** 四段式。
- **输出学习法**：让她用自己的话讲一条线索（如"唐朝由盛转衰怎么发生的"）。
- **理解因果**：多问"为什么会……""带来什么影响"，不死记结论。

## 三.5、命题新方向落点（2026·据济南中考新政）

> 2026 起济南改"综合评价招生"，等级科（含历史）命题转向"少背多想、长材料、跨学科、重过程"。下面把四个方向翻译成**历史可操作的练法**，复习陪练与出卷都按这四条落地。

### ① 减记忆、增高阶 —— 别只背"是什么"，练"为什么/比较/评价"
- 每过一个考点，逼她追问三件事：**为什么会发生**（背景因果）、**和谁像/不像**（横向比较）、**该怎么评价**（一分为二）。
- 比较类设问当家常练：澶渊之盟 vs 宋金和议、岁币 vs 岁赐、贞观之治 vs 开元盛世、废丞相 vs 设军机处、坊市制 vs 宋代商业、郑和下西洋 vs 闭关锁国。让她**列两栏对比表**（相同点/不同点），而不是各背一遍。
- 评价类一律"**一分为二**"：如大运河（沟通南北 vs 劳民伤财）、王安石变法（理财强兵 vs 用人不当）、闭关锁国（暂御外侵 vs 落后挨打）。

### ② 长情境先提取 —— 先读材料找信息，再答题
- 现在材料题题干越来越长（史料原文、诗句、地图、表格、生活/科技情境）。教她**两步走**：先用笔在材料里**圈关键词**（时间、人物、地点、动词、转折词"然/但/故"），再回头答。
- 专项练"**从材料读出信息**"：给她一段她没背过的史料，只问"这段话讲的是哪件事/哪个朝代/什么态度"，训练提取能力，而不是直接默写课本。
- 提醒她：**材料题答案先来自材料，再补课本**，绝不抛开材料空背知识点（这是头号失分点，见第四节）。

### ③ 跨学科 + 家国情怀 —— 与语文/地理交叉，落价值引领
- **跨语文**：用诗文当史料（如皮日休《汴河怀古》评大运河、杜甫诗看安史之乱），让她读懂诗背后的史实与作者态度。
- **跨地理**：大运河连了哪几大水系、海上丝路的港口方位、行省/边疆机构的地理位置——把历史事件**落到地图方位**上。
- **家国情怀/民族共同体**：台湾自古是中国领土、西藏新疆的中央管辖、各民族交融形成中华民族——这类题要会用"**自古以来、中央管辖、共同缔造**"的语言点出**国家统一与民族团结**的价值（呼应会考"民族共同体意识"导向）。

### ④ 重过程规范 —— 论从史出、分点作答、史论结合
- 材料分析题的"得分公式"教给她，落到笔头：
  - **论从史出**：先"据材料……（引材料原话/信息）"，再"可见/说明……（得出结论）"，**有史才有论，不空喊结论**。
  - **分点作答**：用"①②③"分点，一点一句，**踩点给分**；切忌一大段糊在一起。
  - **史论结合**：观点 + 史实支撑，两者都要有（只摆观点没史实、或只罗列史实没观点都扣分）。
- 答题套用第三节四段式/四方向模板时，**显式写出层次词**（背景/措施/作用/认识；东南/西南/西北/东北），让阅卷一眼看到采分点。

## 四、高频坑（优先攻）
- **朝代顺序**记乱。
- **相近事件/人物混**（如丝绸之路路线顺序、各政权对应民族）。
- **时间易混点**（郑成功收台 1662 ≠ 清设台湾府 1684）。
- **材料题脱离材料空答**（头号失分点）：背了课本却不引材料 → 练"据材料……可见……"。
- **比较题只各背一遍、不答异同**：要列"相同点/不同点"两栏，直接回应"比较"二字。
- **结论无史实、史实无结论**：论从史出两头都要有（史论结合）。
- **大段糊写、不分点**：材料题一律①②③分点踩点。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按单元过时间线+核心事件，合书复述一条线索。
2. **专项突破**：错题归因，专攻易混事件/政权表/配对题。
3. **限时模拟**：限时整卷，重点练材料分析题。

## 六、当堂自测形式（交给 `exam-maker`）
政权并立填表、人物事件配对、**长材料解读（先提取后作答）**、**比较类简答（列异同）**、**因果/影响分析**、**评价类（一分为二）**；按 2026 方向少出纯年代识记，多出高阶设问，材料题示范"论从史出·分点·史论结合"的规范写法。

## 七、闭环沉淀
错题入 `错题本/`（易混事件、配错的人物单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
