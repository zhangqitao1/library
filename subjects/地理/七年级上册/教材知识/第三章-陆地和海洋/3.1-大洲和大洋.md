# 大洲和大洋

## 概念 / 定义
地球表面其实是个"**水球**"：海洋多、陆地少。陆地被分成**七大洲**，海洋被分成**四大洋**。这一节要把它们的**名字、大小排序、位置、分界线**记牢。

## 公式 · 规则 · 结论（要能直接背）
**① 海陆比例**
- **"七分海洋，三分陆地"**：海洋约占 **71%**，陆地约占 **29%**。
- 无论怎么平分地球，**任何半球都是海洋多于陆地**。

**② 七大洲（按面积由大到小）**
口诀：**亚非北南美，南极欧大洋**
1. 亚洲（最大）2. 非洲 3. 北美洲 4. 南美洲 5. 南极洲 6. 欧洲 7. 大洋洲（最小）
- 跨纬度最广、跨经度最广、面积最大 → **亚洲**。
- **全部在南半球**的洲：南极洲（也是平均海拔最高、冰雪覆盖）。
- 赤道横穿的大洲：非洲、亚洲、大洋洲、南美洲。

**③ 四大洋（按面积由大到小）**
口诀：**太大印北**
1. **太平洋**（最大最深）2. **大西洋**（呈"S"形）3. **印度洋** 4. **北冰洋**（最小最浅、纬度最高）

**④ 重要大洲分界线（高频）**
- **亚洲 / 欧洲**：乌拉尔山—乌拉尔河—里海—大高加索山—黑海—土耳其海峡。
- **亚洲 / 非洲**：**苏伊士运河**。
- **南美洲 / 北美洲**：**巴拿马运河**。
- **亚洲 / 北美洲**：白令海峡。

**⑤ 陆地几个名词**
- **大陆**：面积广大的陆地（如亚欧大陆，最大）。**岛屿**：小块陆地（格陵兰岛最大）。
- **半岛**：三面环水、一面连陆（如阿拉伯半岛最大）。**大洲 = 大陆 + 附近岛屿**。

## 典型例题（带完整步骤）
> 例：苏伊士运河和巴拿马运河分别是哪两个大洲的分界线？
>
> 解：**苏伊士运河**——连接地中海和红海，是**亚洲与非洲**的分界；**巴拿马运河**——是**南、北美洲**的分界。
> 记忆抓手：苏伊士在埃及（非洲东北），巴拿马在中美洲（连南北美）。

## ⚠️ 易错点 · 陷阱
- 🔴 七大洲面积排序口诀"**亚非北南美，南极欧大洋**"，亚洲最大、大洋洲最小。
- 🔴 苏伊士运河（亚/非）vs 巴拿马运河（南/北美），两条运河别记反。
- **大洋洲 ≠ 澳大利亚**：大洋洲是一个洲（含澳大利亚大陆+众多岛屿），澳大利亚是其中的国家/大陆。
- 北冰洋是**最小、最浅、纬度最高**的洋，不是北大西洋的一部分。
- 南极洲全在南半球、几乎全被冰雪覆盖，但它**不是大洋**而是大洲。
