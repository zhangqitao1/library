# 世界的气候

## 概念 / 定义
把一个地方**长期的气温和降水**综合起来，就是它的**气候类型**。世界主要气候类型有十几种，要会**结合气温曲线+降水柱状图判断是哪种气候**——这是七上地理的"压轴技能"。

## 公式 · 规则 · 结论（要能直接背）
**① 主要气候类型与分布（按温度带记）**
| 温度带 | 气候类型 | 特点 | 主要分布 |
|---|---|---|---|
| **热带** | 热带雨林气候 | **全年高温多雨** | 赤道附近（亚马孙、刚果） |
| | 热带草原气候 | 全年高温，**分干湿两季** | 雨林两侧（非洲最广） |
| | 热带季风气候 | 全年高温，**旱雨两季**（雨季集中） | 印度半岛、中南半岛 |
| | 热带沙漠气候 | **全年高温少雨** | 回归线附近大陆西岸（撒哈拉） |
| **温带** | 亚热带季风气候 | 夏热多雨、冬温少雨 | 我国南方 |
| | 地中海气候 | **夏季炎热干燥、冬季温和多雨**（雨热不同期） | 地中海沿岸 |
| | 温带季风气候 | 夏热多雨、冬冷干燥 | 我国北方（含济南附近） |
| | 温带大陆性气候 | 冬冷夏热、降水少、**温差大** | 亚欧大陆内部 |
| | 温带海洋性气候 | **全年温和湿润** | 西欧 |
| **寒带** | 寒带气候 | 全年严寒 | 两极地区 |
| 高原山地 | 高原山地气候 | 气温随高度变化 | 青藏高原等 |

**② 判读气候类型"三步法"（核心）**
1. **以温定带**：看最冷月气温——最冷月>15℃ 是**热带**；0~15℃ 多是**亚热带/温带海洋**；<0℃ 是**温带**；终年低温是**寒带**。
2. **以水定型**：再看降水——年雨型/夏雨型/冬雨型/少雨型，定具体气候。
3. 综合定名（如"夏雨"+"温带"=温带季风）。

**③ 影响气候的三大因素**
- **纬度位置**（决定冷热）；**海陆位置**（决定干湿、海洋性/大陆性）；**地形地势**（高处冷、迎风坡湿）。

## 典型例题（带完整步骤·气候判读示范）
> 例：某地最冷月均温5℃，夏季干燥少雨、冬季温和多雨。是什么气候？
>
> 解：**第一步以温定带**：最冷月5℃（0~15℃）→ 亚热带。
> **第二步以水定型**：**冬季多雨、夏季干燥**（雨热不同期）→ 这是**地中海气候**的标志。
> 答：地中海气候，分布在地中海沿岸（南北纬30°~40°大陆西岸）。

## ⚠️ 易错点 · 陷阱
- 🔴 **地中海气候 = 夏干冬雨**（雨热不同期），是唯一"冬季多雨"的代表，最爱考。
- 🔴 判读用"**先看气温定温度带，再看降水定类型**"，顺序别乱。
- 热带的四种气候都**全年高温**，区别在降水（雨林=年多雨、沙漠=年少雨、草原/季风=分干湿）。
- 季风气候**雨热同期**（夏季又热又多雨）；地中海**雨热不同期**。
- 气候 ≠ 天气（接 4.1）；气候是长期综合。

## 变式 · 拓展 · 会考链接
- 会考**压轴**常给"气温曲线+降水柱状图"判气候类型并说分布、说成因。
- 联系生活：不同气候下的民居、衣着、农业（如热带雨林的高脚屋）。
- 与 4.2、4.3 三节合起来是一个完整知识链。

## 一问一答（AI 复习时优先用这些来问她）
- Q: 判断气候类型的"三步法"是什么？ ｜ A: 先以温定带（看气温），再以水定型（看降水），最后综合定名。
- Q: 地中海气候的最大特点是什么？ ｜ A: 夏季炎热干燥、冬季温和多雨（雨热不同期）。
- Q: 全年高温多雨是什么气候？ ｜ A: 热带雨林气候。
- Q: 影响气候的三大因素是什么？ ｜ A: 纬度位置、海陆位置、地形地势。
- Q: 季风气候和地中海气候在"雨热"上有什么不同？ ｜ A: 季风雨热同期，地中海雨热不同期。
