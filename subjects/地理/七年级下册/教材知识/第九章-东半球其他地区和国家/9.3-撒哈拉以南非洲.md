# 撒哈拉以南非洲

## 概念 / 定义
撒哈拉以南非洲是**黑种人的故乡**，有"热带大陆"之称。这里**热带草原气候**广布，野生动物资源丰富，但面临**人口、粮食、环境**三大问题，经济以**单一商品经济**为主，发展任务艰巨。

## 公式 · 规则 · 结论（要能直接背）
**① 位置与范围**
- 位于**撒哈拉沙漠以南**，东临印度洋，西临大西洋。
- 赤道横穿中部，绝大部分位于**热带**。
- 海岸线较平直，少优良港湾。

**② "黑种人的故乡"**
- 该地区 **90% 以上**人口为**黑色人种**（⚠️ 以教材数据为准）。
- 文化多样，能歌善舞，击鼓等传统文化发达。

**③ 气候与景观**
- 以**热带草原气候**为主（赤道附近为热带雨林气候，南北两端为热带沙漠气候）。
- 热带草原：**全年高温，分干湿两季**。
- 湿季草木茂盛，干季草木枯黄 → 动物迁徙（斑马、长颈鹿、狮子等）。
- 被称为**"热带大陆"**。

**④ "人口、粮食、环境"问题（PPE 问题）**
- **人口增长快** → 粮食需求大增。
- **农业生产落后** + 频繁旱灾 → **粮食短缺**。
- 过度开垦、过度放牧 → **土地荒漠化** → 生态环境恶化。
- 形成恶性循环：人口增→毁林开荒→环境差→更穷→人口再增……

**⑤ 经济特征**
- 历史上遭受殖民统治，经济畸形发展。
- **单一商品经济**：出口一种或几种初级农矿产品（可可、咖啡、铜、金刚石等），进口工业制成品 → **在国际贸易中处于不利地位**。
- 努力发展民族经济、多样化农业和制造业。

## 典型例题（带完整步骤）
> 例：撒哈拉以南非洲人口、粮食、环境问题之间有什么关系？
>
> 解：**人口增长快** → 需要更多粮食 → **过度开垦、放牧**破坏植被 → **土地荒漠化、旱灾加重** → 粮食更少、更穷 → 人口压力更大 → 恶性循环。

## ⚠️ 易错点 · 陷阱
- 🔴 撒哈拉以南非洲 ≠ 整个非洲（北非以阿拉伯人、白种人为主，属西亚北非文化圈）。
- 🔴 主要气候是**热带草原**（干湿季），不是全年多雨的热带雨林。
- 单一商品经济 = 出口**初级产品**，不是出口汽车飞机。
- "黑种人故乡"是文化/人种概念，不是说只有这一种人。
- 解决问题方向：控制人口、发展粮食生产、保护环境，不能只答"多生孩子"。
