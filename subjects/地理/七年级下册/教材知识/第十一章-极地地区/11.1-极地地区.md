# 极地地区

## 概念 / 定义
极地地区是地球上**最寒冷**的地方，分为**南极**和**北极**两部分。这里自然环境极端，但科学考察价值巨大，也是全球气候变化最敏感的区域，需要全人类共同保护。

## 公式 · 规则 · 结论（要能直接背）
**① 范围（两个地区别搞混）**
- **南极地区**：**南极圈（66.5°S）以南**，包括**南极洲**及其周边海域。
- **北极地区**：**北极圈（66.5°N）以北**，包括**北冰洋**大部分及周边亚、欧、北美三洲的陆地和岛屿。
- 🔑 **南极地区 ≠ 南极洲**：南极地区包括南极洲+周边海域；南极洲指南极大陆及周边岛屿。

**② 自然环境对比**

| 比较项 | 南极地区 | 北极地区 |
|---|---|---|
| 主体 | 南极大陆（冰雪高原） | 北冰洋（海洋为主） |
| 气温 | 更冷（地球上最冷） | 相对暖和些 |
| 降水 | 极少（"白色荒漠"） | 比南极多 |
| 风速 | 极大（"风库"） | 远小于南极 |
| 代表动物 | **企鹅** | **北极熊** |
| 居民 | 无常住人口（科考人员） | 有因纽特人等原住民 |

- 南极三个称号：**冰雪高原、白色荒漠、风库**。
- 南极独特景观：**冰山**（冰川崩裂入海形成）。

**③ 科学考察**
- 南极：天然实验室（冰川、气候、天文、生物等研究）。
- 资源：矿产、**固体淡水**（冰层）、海洋生物。
- **最佳科考时间——暖季**：
  - 南极暖季：**11月—次年3月**（相对暖和，有极昼）。
  - 北极暖季：**6—8月**。
- 中国南极科考站：长城站、中山站、昆仑站、泰山站、**秦岭站**（⚠️ 2024年开站，以教材为准）。
- 中国北极科考站：**黄河站**。

**④ 环境保护**
- 问题：全球变暖导致冰川融化、海平面上升；过度捕杀极地动物；环境污染。
- 《南极条约》等：和平利用南极，禁止军事化，保护生态环境。
- 北极：油气开发带来污染风险。

## 典型例题（带完整步骤）
> 例：为什么南极科考选在 11 月到次年 3 月？
>
> 解：这是南极的**暖季** → 气温相对较高（仍很冷）、有**极昼**（白昼长，便于作业）、冰雪相对稳定 → 相对安全，适合考察。相反 4—10 月寒季极端寒冷、极夜，不宜考察。

## ⚠️ 易错点 · 陷阱
- 🔴 **企鹅在南极，北极熊在北极**——绝对不要搞反（经典送分/丢分题）。
- 🔴 南极比北极**更冷**（南极是陆地冰雪高原，北极是海洋为主）。
- 🔴 南极暖季是**11月—次年3月**（北半球冬天时），别用北半球季节去推。
- 南极洲是**跨经度最广**的大洲（所有经线交汇），不是跨纬度最广（亚洲）。
- 长城站位置特殊：在南极圈**以外**（⚠️ 教材图示），无极昼极夜现象。
- 秦岭站为新版教材新增站名，旧资料可能没有。
