# 美国

## 概念 / 定义
美国是西半球最大的发达国家，是**移民国家**——各民族大熔炉。它农业高度发达且**地区专业化生产**，高新技术产业在世界领先，硅谷是信息技术中心。

## 公式 · 规则 · 结论（要能直接背）
**① 位置与领土**
- 位于**北美洲中部**，东临大西洋，西临太平洋。
- 领土由三部分组成：
  - **本土**（48个州+首都华盛顿所在的哥伦比亚特区）
  - **阿拉斯加**（北美洲西北部）
  - **夏威夷**（太平洋上的群岛）
- 地跨**热带、北温带、北寒带**（夏威夷在北回归线附近，阿拉斯加有部分在北极圈内）。

**② 移民国家**
- 除原住民印第安人外，多为**欧洲、亚洲、非洲、拉丁美洲**移民后裔。
- 民族构成复杂，被称为**"民族大熔炉"**。
- 华人和华侨较多，在旧金山、纽约、洛杉矶等城市。

**③ 农业——地区专业化生产**
- 农业高度发达，**机械化、专业化、商品化**程度高。
- **农业带/区**（读图必会）：
  - 玉米带、小麦区、棉花带、乳畜带、亚热带作物带等。
- 每个地区根据自然条件专注生产一种或几种农产品 → **地区专业化**。
- 农产品大量出口。

**④ 工业与高新技术产业**
- 工业体系完整，技术先进。
- **高新技术产业**发达：信息技术、生物技术、新材料等。
- **圣弗朗西斯科（旧金山）东南的"硅谷"**是世界著名的信息技术产业中心。
- 传统工业向西部和南部迁移（⚠️ 教材表述）。

**⑤ 自然与资源**
- 地形：西部高山（落基山脉）、中部平原、东部低山高原。
- 密西西比河是重要内河航道。
- 自然资源丰富，耕地面积广大。

> 标本土、阿拉斯加、夏威夷 → 说明美国是**地跨三带的移民大国**。
>

> 步骤：中部平原找玉米带、小麦区 → 南部找棉花带 → 东北部找乳畜带 → 归纳"因地制宜、地区专业化"。
>
> **示范3：读美国主要工业和高新产业分布图**
> 找到硅谷（旧金山附近）→ 纽约、芝加哥等传统工业城市 → 对比东海岸与西海岸产业。

## 典型例题（带完整步骤）
> 例：美国农业为什么要实行地区专业化生产？
>
> 解：读农业带图 → 美国国土辽阔、自然条件多样 → 不同地区气温、降水、土壤不同 → 因地制宜只种最适合的作物 → **机械化大规模生产** → 效率高、成本低 → 形成玉米带、乳畜带等专业化产区。

## ⚠️ 易错点 · 陷阱
- 🔴 美国领土**三部分**，别只画本土忘了阿拉斯加和夏威夷。
- 🔴 首都是**华盛顿**，不是纽约（纽约是最大城市之一）。
- 🔴 硅谷在**旧金山（圣弗朗西斯科）**附近，不在纽约或洛杉矶。
- 农业专业化 ≠ 只种一种作物全国（是**不同地区**专门种不同东西）。
- 美国是发达国家，农业却很强（不要觉得发达国家不种地）。
