# 地理七下 · 高频易错点汇总

## 第七章 亚洲
- 🔴 **"亚洲全部在东/北半球"错**——主体在，有小部分跨界。
- 地势**中部高四周低** → 河流**放射状**外流，不是都向东流。
- 亚洲六大分区：东亚、东南亚、南亚、西亚、北亚、中亚——后面各国要能对上号。
- 人口过亿 **7 国**：中印印尼巴孟日菲，别漏孟加拉国、菲律宾。
- 里海是**湖**不是海，但是世界最大湖泊。

## 第八章 邻近地区和国家
- 🔴 日本：**加工贸易**经济，工业在**太平洋沿岸**（不是内陆）；多火山地震因**板块交界**。
- 🔴 马六甲海峡在**马来半岛与苏门答腊岛之间**，不是苏伊士运河。
- 中南半岛地形：**山河相间、纵列分布**（不是中间高四周低）。
- 🔴 印度：**热带季风**气候，**西南季风**带来雨季（不是东北季风）；水旱灾害因季风**不稳定**。
- 湄公河在中国叫**澜沧江**。
- 俄罗斯：**面积最大**、跨亚欧，但首都莫斯科在**欧洲部分**；交通以**铁路**为主。
- 新加坡是**城市国家**，不是马来西亚的一部分。

## 第九章 东半球其他地区和国家
- 🔴 西亚"**五海三洲**"：里海、黑海、地中海、红海、阿拉伯海 + 亚非欧交界。
- 石油主要在**波斯湾沿岸**，不是地中海沿岸。
- 欧洲西部：**温带海洋性**→畜牧业；**地中海气候**→夏干冬雨（别和季风"雨热同期"混）。
- 埃菲尔铁塔在**法国**，斗兽场在**意大利**。
- 撒哈拉以南非洲 ≠ 整个非洲；主要气候是**热带草原**（干湿季）。
- **单一商品经济**=出口初级农矿产品，在国际贸易中处不利地位。
- 澳大利亚：**独占大陆**、**南半球发达国家**；首都**堪培拉**（不是悉尼）；两个称号别混。
- 澳大利亚特有生物因**长期孤立演化**，不是"因为热"。

## 第十章 西半球的国家
- 🔴 美国领土**三部分**：本土 + 阿拉斯加 + 夏威夷。
- 首都**华盛顿**，不是纽约；**硅谷在旧金山**附近，不在纽约。
- 农业是**地区专业化**（玉米带、乳畜带等），不是全国只种一种。
- 🔴 巴西首都**巴西利亚**（不是里约）；说**葡萄牙语**（拉美唯一葡语大国）。
- 亚马孙雨林在**巴西北部**；保护原因答**生态意义**（地球之肺），不只说"树多"。

## 第十一章 极地地区
- 🔴 **企鹅=南极，北极熊=北极**——绝对别搞反！
- 南极比北极**更冷**；南极是陆地冰雪，北极以海洋为主。
- 🔴 南极暖季 = **11月—次年3月**（北半球冬天时去考察），别用本地季节推。
- 南极地区 ≠ 南极洲（地区含周边海域）。
- 南极科考站：**秦岭站**为2024新版新增（⚠️ 以教材为准）；长城站在南极圈外无极昼极夜。

## 通用区域地理读图方法（七下"答题套路"）
1. **位置三步**：半球（东/西、南/北）→ 纬度带（热/温/寒）→ 海陆位置（临什么洋什么洲）。
2. **气候与农业**：先定气候类型 → 再推农业/物产（季风→水稻；沙漠→灌溉；牧草→畜牧）。
3. **资源与经济**：石油区→西亚；加工贸易→日本；单一商品→撒哈拉以南非洲；矿车/羊背→澳大利亚。
4. **国家定位**：在地图上标出 → 说分区（东亚/东南亚等）→ 说邻海邻国。
5. **对比题**：用表格（位置/气候/经济/问题）——南极vs北极、美国vs巴西、西亚vs东南亚。
