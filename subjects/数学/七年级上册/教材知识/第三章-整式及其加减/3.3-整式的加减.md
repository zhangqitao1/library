# 整式的加减

## 概念 / 定义
整式的加减，本质就是**去括号 + 合并同类项**。最后把式子化到"最简"——没有同类项、没有括号。一句话：**只有同类项才能合并，合并就是把系数加减、字母部分不变**。

## 公式 · 规则 · 结论（要能直接背）
- **合并同类项**：系数相加减，**字母和指数都不变**。如 3a + 5a = 8a；4x²−7x² = −3x²。
- **去括号法则**（重点，符号最易错）：
  - 括号前是 **"+"**：去掉括号，里面**各项符号不变**。a + (b − c) = a + b − c。
  - 括号前是 **"−"**：去掉括号，里面**各项都要变号**。a − (b − c) = a − b + c。
  - 括号前有**系数**：要**乘到每一项**，再按符号法则。−2(x − 3) = −2x + 6。
- **添括号法则**（去括号的逆操作）：
  - 添"+"括号，括进去各项不变号；添"−"括号，括进去各项都变号。
- 整式加减步骤：**①去括号 → ②找同类项 → ③合并 → ④结果按次数从高到低排**。

## 典型例题（带完整步骤）
> 例 1：化简 3(2a − b) − 2(a − 3b)。
>
> 解：
> 第 1 步：**去括号**，把系数乘进每一项：
> 3(2a−b) = 6a − 3b；−2(a−3b) = −2a + 6b（注意 −2×(−3b)=+6b）。
> 式子 = 6a − 3b − 2a + 6b。
> 第 2 步：**找同类项**：6a 与 −2a 是一类；−3b 与 +6b 是一类。
> 第 3 步：**合并**（系数加减）：(6−2)a + (−3+6)b = **4a + 3b**。
> ✅ 检验：用 a=1,b=1 验：原式 3(2−1)−2(1−3)=3+4=7；结果 4+3=7，一致。
>
> 例 2（化简求值）：先化简 2(x²−x) − (3x² − 2x) ，再求当 x = −1 时的值。
>
> 解：
> 第 1 步：去括号：2x² − 2x − 3x² + 2x（括号前是 −，里面各项变号）。
> 第 2 步：合并同类项：(2−3)x² + (−2+2)x = **−x²**（x 项抵消为 0）。
> 第 3 步：代入 x = −1：−x² = −(−1)² = −(1) = **−1**。
> ⚠️ 提醒：先化简再代入，省去大量计算；代入时 (−1)² 要加括号。

## ⚠️ 易错点 · 陷阱
- **去括号"前面是减号"忘了变号**——头号失分点。a − (b − c) = a − b + c，每一项都变。
- **括号前系数没乘到每一项**：−2(x−3) 要等于 −2x+6，不是 −2x−3。
- **只把第一项变号**：−(a−b+c) = −a+b−c，三项全变。
- **不是同类项硬合并**：3a + 2b 不能合成 5ab，只能留着。
- 化简求值题：**先化简再代入**，别一上来就代入硬算。

## 变式 · 拓展 · 中考链接
- 去括号、合并同类项是**解方程"去括号、移项"**的同款技能，符号错则方程全错。
- 中考化简求值题常用"整体代入"或"先化简"技巧，分值稳定。
- "求两个多项式的和/差"也是先列式、再去括号合并。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 括号前是减号，去括号时里面怎么变？ ｜ A: 每一项都变号。
- Q: 3a + 2b 能合并成 5ab 吗？ ｜ A: 不能，它们不是同类项。
- Q: −2(x−3) 去括号等于什么？ ｜ A: −2x+6。
- Q: 合并同类项时字母和指数变不变？ ｜ A: 不变，只把系数相加减。
- Q: 化简求值题应该先做什么？ ｜ A: 先化简，再代入数值。
