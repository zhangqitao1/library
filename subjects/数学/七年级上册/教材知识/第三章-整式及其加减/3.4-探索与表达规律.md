# 探索与表达规律

## 概念 / 定义
这一节是把前面"用字母表示数"用起来：观察一列数或一组图形，找出它们**怎么一个一个变过来的**，然后用含 n 的代数式表示出**第 n 个**是多少。核心思想：**找出"序号 n"和"结果"之间的关系**。

## 公式 · 规则 · 结论（要能直接背）
- **找规律三步**：①写出前几项与序号对照；②看相邻两项**差多少/乘几倍**；③用 n 表示通式，再**代入验证**。
- 常见规律模型：
  - **等差型**（每次加同一个数 d）：第 n 个 = **首项 + (n−1)·d**。如 2,5,8,11… → 3n−1。
  - **平方型**：1,4,9,16… → **n²**。
  - **奇数/偶数**：偶数 **2n**，奇数 **2n−1**。
  - **正负交替**：用 **(−1)ⁿ**（n 偶为正）或 **(−1)ⁿ⁺¹**（n 奇为正）控制符号。
- **图形题**：把"第 n 个图形里有几根火柴/几个点"数清前 3 个，找出**每多一个图增加几个**（就是 d），套等差公式。

## 典型例题（带完整步骤）
> 例：用火柴棒搭三角形，第 1 个用 3 根，第 2 个用 5 根，第 3 个用 7 根……
> （1）第 n 个用多少根？（2）第 100 个用多少根？
>
> 解：
> 第 1 步：**列对照表**：序号 1,2,3 → 火柴 3,5,7。
> 第 2 步：看变化——每多搭一个三角形，**多用 2 根**（5−3=2，7−5=2），所以 d = 2。
> 第 3 步：套等差公式 第 n 个 = 首项 + (n−1)·d = 3 + (n−1)×2 = 3 + 2n − 2 = **2n + 1**。
> 第 4 步：**验证**：n=1 → 2+1=3 ✓；n=2 → 5 ✓；n=3 → 7 ✓。
> （2）第 100 个：把 n=100 代入 2n+1 = 2×100+1 = **201 根**。
> ✅ 检验：通式经过前三项验证，代入放心。

## ⚠️ 易错点 · 陷阱
- **得出通式后一定要代 n=1,2,3 验证**，不验证常把公式写偏。
- **等差公式是 (n−1)·d 不是 n·d**：第 1 个时增量为 0。
- **正负交替忘了用 (−1)ⁿ**：判断 n 取奇/偶时正负，先看第 1 项是正是负来选 (−1)ⁿ 还是 (−1)ⁿ⁺¹。
- 图形题**数错初始几项**导致全错，先稳稳数对前 3 个。
- "第 n 个"求出的是**通式**，"第 100 个"是把 100 代进去，别混。

## 变式 · 拓展 · 中考链接
- 中考压轴常考"找规律 + 代数式"，从数字串、图形点阵、数表（杨辉三角风格）出题。
- 进阶：求"前 n 个的和"或"第 n 行第 m 个"，本质还是先找单变量规律。
- 是用字母表示数、代数式求值能力的综合应用。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 找规律的通式写出来后，第一件事该做什么？ ｜ A: 代入 n=1,2,3 验证。
- Q: 2,5,8,11… 第 n 个怎么表示？ ｜ A: 3n−1。
- Q: 偶数用字母怎么表示？奇数呢？ ｜ A: 偶数 2n，奇数 2n−1。
- Q: 等差规律第 n 项公式是什么？ ｜ A: 首项 + (n−1)×公差。
- Q: 数列符号正负交替，用什么表示？ ｜ A: (−1)ⁿ 或 (−1)ⁿ⁺¹。
