# 正负数与有理数

## 概念 / 定义
**正数**就是大于 0 的数（如 3、0.5）；**负数**是在正数前加"−"号、小于 0 的数（如 −2、−1.5）。负数用来表示**和正数意义相反**的量（赚 5 元 / 亏 −5 元）。**0 既不是正数也不是负数**，是正负的分界。

## 公式 · 规则 · 结论（要能直接背）
- **相反意义的量**：规定一个方向为正，另一个就是负。例：向东 +3 米，向西 −3 米；零上 +5℃，零下 −5℃。
- **有理数 = 整数 + 分数**（凡是能写成"两个整数相除"分数形式的数）。
  - 整数：正整数、0、负整数。
  - 分数：正分数、负分数（有限小数、无限循环小数都算分数）。
- 另一种分法：有理数 = **正有理数 + 0 + 负有理数**。
- **0 的特殊身份**：是整数、是偶数、是自然数；不是正数也不是负数；是最小的非负数。
- 易混点：**π = 3.1415926…（无限不循环）不是有理数**；但课本七上还没正式讲无理数，知道"π 不能写成分数"即可。

## 典型例题（带完整步骤）
> 例：把下列各数填入对应集合：−3，+2.5，0，−4/7，6，−1.2，100%（即 1）。
> ① 正数集合 ② 负数集合 ③ 整数集合 ④ 分数集合
>
> 解：
> 第 1 步：先逐个判断正负。正数：+2.5、6、1（=100%）；负数：−3、−4/7、−1.2；0 不进正负集合。
> 第 2 步：判断整数还是分数。整数（不含小数部分）：−3、0、6、1；分数（含小数/分数形式）：+2.5、−4/7、−1.2。
> 第 3 步：填集合：
> ① 正数：{ +2.5, 6, 1 }
> ② 负数：{ −3, −4/7, −1.2 }
> ③ 整数：{ −3, 0, 6, 1 }
> ④ 分数：{ +2.5, −4/7, −1.2 }
> ✅ 检验：0 只进整数集合，没进正、负集合，正确。

## ⚠️ 易错点 · 陷阱
- **0 不是正数也不是负数**——这是最常考的"陷阱判断题"。
- **"带负号的不一定是负数"**：如 −a，当 a 是负数时 −a 反而是正数。看到字母别想当然。
- 有限小数、循环小数都是**分数（有理数）**；只有无限不循环（如 π）不是。
- "非负数"= 0 和正数；"非正数"= 0 和负数。看到"非"字要把 0 算进去。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 0 是正数吗？是整数吗？ ｜ A: 0 不是正数也不是负数，但它是整数。
- Q: 如果向东走记作正，那向西走 5 米怎么记？ ｜ A: −5 米。
- Q: −0.333…（无限循环）是有理数吗？ ｜ A: 是，循环小数能写成分数。
- Q: 有理数按"整数、分数"怎么分？ ｜ A: 整数（正整数、0、负整数）+ 分数（正分数、负分数）。
