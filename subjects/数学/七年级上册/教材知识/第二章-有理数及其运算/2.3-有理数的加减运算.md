# 有理数的加减运算

## 概念 / 定义
有理数加减，关键是**先定符号，再算绝对值**。减法没有单独的"减"，而是**减去一个数 = 加上它的相反数**，统一转成加法来做。

## 公式 · 规则 · 结论（要能直接背）
- **加法法则**（口诀：同号相加、异号相减、绝对值定符号）：
  - **同号两数相加**：取相同的符号，绝对值**相加**。(−3)+(−5) = −8。
  - **异号两数相加**：取**绝对值大**的那个数的符号，用大绝对值**减**小绝对值。(−7)+3 = −4；7+(−3) = +4。
  - 一个数加 0 还得这个数：a + 0 = a。
  - 互为相反数的两数相加得 0。
- **减法法则**（必背）：**a − b = a + (−b)**。减去一个数，等于加上这个数的相反数。
  - 例：3 − (−5) = 3 + 5 = 8；−2 − 6 = −2 + (−6) = −8。
- **加法运算律**（让计算更快）：
  - 交换律：a + b = b + a；
  - 结合律：(a + b) + c = a + (b + c)。
  - 技巧：把**同号的凑一起**、把**互为相反数/能凑整的凑一起**先算。

## 典型例题（带完整步骤）
> 例：计算 (−12) + 8 − (−5) + (−3)。
>
> 解：
> 第 1 步：**先把减法全转成加法**（减法法则）：
> = (−12) + 8 + (+5) + (−3)。  ← 减(−5)变成加(+5)
> 第 2 步：用交换律、结合律**把同号的凑一起**：
> = [(−12) + (−3)] + [8 + 5]
> = (−15) + 13。  ← 同号相加：负的凑成 −15，正的凑成 13
> 第 3 步：**异号相减**，取绝对值大的符号：|−15|=15 > |13|=13，符号取负：
> = −(15 − 13) = **−2**。
> ✅ 检验：从左到右逐步算 (−12)+8=−4，−4−(−5)=1，1+(−3)=−2，一致。

## ⚠️ 易错点 · 陷阱
- **减号、负号分不清**：a − b = a + (−b)，"减去负数 = 加上正数"，3 − (−5) 要变成 3 + 5，最易错。
- **异号相加时符号取错**：要取**绝对值大**的那个数的符号，不是随便取。
- **连减/连加混合**：先全部统一成加法，再凑同号，能大幅减少出错。
- 0 减一个数：0 − 5 = −5，0 − (−5) = 5，别漏符号。

## 变式 · 拓展 · 中考链接
- 加减混合是后面**整式加减、解方程移项**的基础，符号错了后面全错。
- 常考"绝对值 + 加减"综合：先算绝对值再加减。
- 实际问题：温差（最高 − 最低）、账本结余、海拔升降。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 减去一个数等于加上什么？ ｜ A: 加上这个数的相反数。
- Q: (−6) + (−4) 等于几？用了什么法则？ ｜ A: −10，同号相加取同号、绝对值相加。
- Q: (−7) + 3 的符号怎么定？ ｜ A: 取绝对值大的 −7 的符号，结果是 −4。
- Q: 5 − (−8) 怎么算？ ｜ A: = 5 + 8 = 13。
- Q: 为什么算加减时要先把减法转成加法？ ｜ A: 统一成加法后能用交换律、结合律凑数，更快更不易错。
