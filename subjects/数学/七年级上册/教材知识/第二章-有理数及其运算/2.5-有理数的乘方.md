# 有理数的乘方

## 概念 / 定义
**乘方**就是"几个相同的数连乘"的简便写法。aⁿ 表示 n 个 a 相乘，a 叫**底数**，n 叫**指数**，结果叫**幂**。例如 2³ = 2×2×2 = 8。

## 公式 · 规则 · 结论（要能直接背）
- **aⁿ = a×a×…×a（n 个 a）**；底数是被乘的数，指数是个数。
- **负数乘方的符号**（重点）：
  - **负数的偶次幂为正**：(−2)² = 4，(−2)⁴ = 16。
  - **负数的奇次幂为负**：(−2)³ = −8，(−2)⁵ = −32。
  - 口诀：**奇负偶正**（负底数时）。
- **特殊**：
  - 任何非零数的 **0 次幂 = 1**（七上一般先记 a¹=a；0次幂在后续学）。
  - 1 的任何次幂都是 1；−1 的偶次幂=1、奇次幂=−1。
  - 0 的正整数次幂都是 0。
- **括号决定底数**（最易错）：
  - **(−2)⁴**：底数是 −2，= 16。
  - **−2⁴**：底数是 2，先算 2⁴=16 再取负 = **−16**。
  - **(−2/3)²** 底数是 −2/3；**−(2/3)²** 底数是 2/3。
- **科学记数法**：把大数写成 **a × 10ⁿ（1 ≤ |a| < 10）**。如 350000 = 3.5×10⁵（n = 整数位数 − 1）。

## 典型例题（带完整步骤）
> 例 1：计算 (−3)² 与 −3² 的值，并说明区别。
>
> 解：
> 第 1 步：(−3)² 的**底数是 −3**（括号括住了负号），= (−3)×(−3) = **9**。
> 第 2 步：−3² 的**底数是 3**，乘方只管 3，先算 3² = 9，再取负号 = **−9**。
> 结论：(−3)² = 9，−3² = −9，**括号在不在，结果差一个符号**。
>
> 例 2：用科学记数法表示 1 200 000，并写出 (−2)³ × (−1)⁴ 的值。
>
> 解：
> 第 1 步：1 200 000 小数点向左移动 6 位 → **1.2 × 10⁶**（验证：整数 7 位，n = 7−1 = 6）。
> 第 2 步：(−2)³ = −8（奇次幂为负）；(−1)⁴ = 1（偶次幂为正）。
> 第 3 步：(−8) × 1 = **−8**。
> ✅ 检验：负号个数——(−2)³ 含 3 个负号、(−1)⁴ 含 4 个，合计 7 个，奇数 → 负，符合。

## ⚠️ 易错点 · 陷阱
- **(−a)ⁿ 与 −aⁿ 不一样**：前者底数含负号，后者只对 a 乘方再取负。这是头号坑。
- **负数奇偶次幂符号**：奇次幂负、偶次幂正，背熟"奇负偶正"。
- **科学记数法的 a 必须 1≤|a|<10**：35×10⁴ 是错的，应是 3.5×10⁵。
- 指数 n 数错：1 200 000 是 1.2×10⁶ 不是 10⁷（数整数位数再减 1）。
- 分数乘方要给**整个分数**加括号：(2/3)² = 4/9，不是 2/3²。

## 变式 · 拓展 · 中考链接
- 乘方是**混合运算**里"第一优先级"，必须先算。
- 科学记数法在物理、天文数据题里常用，是中考小题常客。
- 为后续"幂的运算（同底数幂相乘）""平方根/立方根"打基础。
