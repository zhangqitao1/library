# 认识一元一次方程

## 概念 / 定义
- **方程**：含有**未知数**的**等式**（必须有等号、必须有字母）。
- **一元一次方程**：只含**一个**未知数（"一元"），并且未知数的**次数是 1**（"一次"），等号两边都是整式。如 2x+3 = 7。
- **方程的解**：能使方程左右两边相等的未知数的值。求解的过程叫**解方程**。

## 公式 · 规则 · 结论（要能直接背）
- 判断一元一次方程**三个条件缺一不可**：①是等式；②只有一个未知数；③未知数次数为 1（且分母里不含未知数）。
  - x² + 1 = 0 不是（二次）；1/x = 2 不是（x 在分母）；2x − 3 不是（没有等号，是代数式）。
- **等式的基本性质（解方程的根本依据）**：
  - **性质 1**：等式两边**同时加或减**同一个数（式），结果仍相等。a=b → a±c = b±c。
  - **性质 2**：等式两边**同时乘**同一个数，或**同时除以同一个非零数**，结果仍相等。a=b → ac=bc；a/c=b/c（c≠0）。
- **验根**：把求得的值代回原方程，左右相等才对。

## 典型例题（带完整步骤）
> 例 1：方程 (m−2)x^|m−1| + 3 = 0 是关于 x 的一元一次方程，求 m。
>
> 解：
> 第 1 步："一次"要求未知数指数 = 1：|m−1| = 1 → m−1 = 1 或 m−1 = −1 → m = 2 或 m = 0。
> 第 2 步："一元一次"要求 x 的系数不为 0：m−2 ≠ 0 → m ≠ 2。
> 第 3 步：综合两步，排除 m=2，得 **m = 0**。
> ✅ 检验：m=0 时方程为 −2x + 3 = 0，确是一元一次方程。
>
> 例 2：检验 x = 3 是不是方程 2x − 1 = 5 的解。
>
> 解：
> 第 1 步：把 x=3 代入左边：2×3 − 1 = 5。
> 第 2 步：左边 = 5 = 右边，相等。
> 结论：x = 3 **是**该方程的解。

## ⚠️ 易错点 · 陷阱
- **"一次"指最高次是 1**，且未知数系数 ≠ 0；判断题里 (m−2)x 型常考"系数不为 0"。
- **分母含未知数的不是一元一次方程**（如 1/x）。
- **没有等号的是代数式不是方程**。
- 等式性质 2 里除数**必须非零**。
- 解出来要**会验根**，尤其应用题，代回原题情境检查合理性。

## 变式 · 拓展 · 中考链接
- 等式两条性质是后面"移项、系数化为 1"的理论根，理解了解方程就不会乱。
- "已知方程的解，反求字母系数"是常考题型：把解代入当作已知，反解字母。
- 是初中所有方程（二元、分式、一元二次）的起点。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 方程必须具备哪两样东西？ ｜ A: 未知数 + 等号（含未知数的等式）。
- Q: "一元一次"分别指什么？ ｜ A: 一个未知数、未知数次数是 1。
- Q: 1/x = 3 是一元一次方程吗？ ｜ A: 不是，x 在分母。
- Q: 等式两边能同时除以 0 吗？ ｜ A: 不能，除数必须非零。
- Q: 怎么检验一个数是不是方程的解？ ｜ A: 代回方程，看左右是否相等。
