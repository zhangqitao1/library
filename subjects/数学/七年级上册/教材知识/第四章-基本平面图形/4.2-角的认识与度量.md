# 角的认识与度量

## 概念 / 定义
**角**由**一个顶点和两条边（射线）**组成，可以看作一条射线绕端点旋转形成的。角的大小**只看张开的程度**，和边画多长无关。

## 公式 · 规则 · 结论（要能直接背）
- **度量单位与进率**：1 周角 = 360°，**1° = 60′（分），1′ = 60″（秒）**。进率是 **60**（像时间），不是 100！
- **角的分类**：
  - 锐角：0° < α < 90°；
  - 直角：= 90°；
  - 钝角：90° < α < 180°；
  - 平角：= 180°（两边成一条直线）；
  - 周角：= 360°。
- **角平分线**：从顶点出发把角分成**相等两份**的射线。若 OC 平分 ∠AOB，则 **∠AOC = ∠BOC = ½∠AOB**，∠AOB = 2∠AOC。
- **度分秒换算**：
  - 大化小（度→分→秒）：**乘 60**。
  - 小化大（秒→分→度）：**除 60**。
- **角的表示**：∠AOB（顶点字母 O 写中间）、∠1、∠α。

## 典型例题（带完整步骤）
> 例 1：把 32.5° 换算成"度分秒"。
>
> 解：
> 第 1 步：整数部分 32° 留着，小数 0.5° 化成分：0.5×60 = 30′。
> 第 2 步：拼起来：32.5° = **32°30′**。
> ✅ 反向检验：30′ ÷ 60 = 0.5°，加回 32° = 32.5°，对。
>
> 例 2：OC 平分 ∠AOB，∠AOB = 76°，OD 平分 ∠BOC，求 ∠AOD。
>
> 解：
> 第 1 步：OC 平分 ∠AOB → ∠BOC = ½×76° = 38°。
> 第 2 步：OD 平分 ∠BOC → ∠BOD = ½×38° = 19°。
> 第 3 步：∠AOD = ∠AOB − ∠BOD = 76° − 19° = **57°**。
> ✅ 检验：∠AOD 也 = ∠AOC + ∠COD = 38° + 19° = 57°，一致。

## ⚠️ 易错点 · 陷阱
- **进率是 60 不是 100**：度分秒像时钟，1°=60′、1′=60″。换算最易错。
- **角的大小与边长无关**：边画得长不代表角大，只看张口。
- **角平分线给的是"相等两份"**，∠AOB = 2 倍小角，别只算一半。
- 角的表示**顶点字母必须在中间**：∠AOB 顶点是 O。
- 钝角范围是 90°~180°（不含端点），平角是恰好 180°，别混。

## 变式 · 拓展 · 中考链接
- 角平分线 + 角的和差是中考几何计算高频，常与"分类讨论（射线在角内/外）"结合。
- 度分秒计算（含退位借位，像时分秒减法）会单独考。
- 是后面相交线、平行线（同位角、对顶角）的基础。
