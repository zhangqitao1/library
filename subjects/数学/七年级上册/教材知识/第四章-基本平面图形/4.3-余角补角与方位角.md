# 余角 · 补角与方位角

## 概念 / 定义
- **互为余角**：两个角的和等于 **90°**，就说它们互余，每个是另一个的"余角"。
- **互为补角**：两个角的和等于 **180°**，就说它们互补，每个是另一个的"补角"。
- 注意：余角、补角是**两个角之间的关系**，和它们的位置无关（不一定挨着）。

## 公式 · 规则 · 结论（要能直接背）
- **互余**：∠1 + ∠2 = 90°，则 ∠2 = **90° − ∠1**。
- **互补**：∠1 + ∠2 = 180°，则 ∠2 = **180° − ∠1**。
- **重要性质（必背、常考）**：
  - **同角（或等角）的余角相等**。
  - **同角（或等角）的补角相等**。
- 只有**锐角才有余角**（因为余角要 90° 以内）；**小于 180° 的角才有补角**。
- **方位角**：以**正北、正南**为基准，写成"北偏东 x°""南偏西 y°"。如"北偏东 60°"指从正北方向**向东转 60°**。正东正西正南正北是特殊方向。

## 典型例题（带完整步骤）
> 例 1：一个角是 35°，求它的余角和补角。
>
> 解：
> 第 1 步：余角 = 90° − 35° = **55°**。
> 第 2 步：补角 = 180° − 35° = **145°**。
> ✅ 检验：35°+55°=90°，35°+145°=180°，都对。
>
> 例 2：∠1 与 ∠2 互补，∠1 与 ∠3 互余，且 ∠2 = 130°，求 ∠3。
>
> 解：
> 第 1 步：∠1 与 ∠2 互补 → ∠1 = 180° − 130° = 50°。
> 第 2 步：∠1 与 ∠3 互余 → ∠3 = 90° − ∠1 = 90° − 50° = **40°**。
> ✅ 检验：∠1+∠3 = 50+40 = 90°，互余成立。

## ⚠️ 易错点 · 陷阱
- **余角是和为 90°，补角是和为 180°**，两个数字千万别记反（补角更大，"补"得更多）。
- 求余角/补角是 **90/180 减去原角**，不是除以 2。
- **钝角没有余角**（余角会是负的）；判断"有没有"先看角的范围。
- 方位角"北偏东 60°"要**从北开始量**，不是从东量；先南北、后偏角。
- "同角的补角相等"是常用推理依据，证明题里别忘了它。

## 变式 · 拓展 · 中考链接
- 余角补角 + 方程：常给"一个角的补角比它的余角的 3 倍多 10°"这类，列方程求角。
- "同角的余角/补角相等"是几何证明的常用桥梁。
- 方位角在中考实际应用题（航行、测量）里频繁出现。
