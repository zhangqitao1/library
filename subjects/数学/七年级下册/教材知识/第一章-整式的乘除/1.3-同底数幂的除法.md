# 同底数幂的除法

## 概念 / 定义
同底数幂相除，相当于"约掉相同个数的底数"：a⁵÷a² 约掉 2 个 a，还剩 3 个 a，所以是 a³。口诀：**同底数幂相除，底数不变，指数相减**。

## 公式 · 规则 · 结论（要能直接背）
- **法则**：aᵐ ÷ aⁿ = aᵐ⁻ⁿ（a≠0，m≥n 为正整数时直接减）。
- **零指数幂**：a⁰ = 1（a≠0）；0⁰ **无意义**。
- **负整数指数幂**：a⁻ⁿ = 1/aⁿ（a≠0，n 为正整数）；如 2⁻³ = 1/8。
- 结果通常写成正指数或分数形式，**分母不能有 0**。

## 典型例题（带完整步骤）
> 例 1：计算 x⁷ ÷ x³。
>
> 解：同底数相除，指数 7−3 = 4，得 **x⁴**。
>
> 例 2：计算 2⁵ ÷ 2⁸。
>
> 解：
> 第 1 步：2⁵⁻⁸ = 2⁻³。
> 第 2 步：2⁻³ = 1/2³ = **1/8**。
>
> 例 3：计算 (3a²b)⁰（a≠0，b≠0）。
>
> 解：任何非 0 数的 0 次幂都是 1，得 **1**（与 a、b 无关）。

## ⚠️ 易错点 · 陷阱
- **指数相减，不是相除**：a⁵÷a² = a³，不是 a^(5/2)。
- **a⁰ = 1 的条件是 a≠0**；0⁰ 无意义。
- 负指数别忘倒数：a⁻² = 1/a²，不是 −a²。
- 底数必须相同；2⁵÷4² 要先化 4=2² 再除。

## 变式 · 拓展 · 中考链接
- 科学记数法小数的幂运算、整式除法中的指数化简都依赖本法则。
- 八年级分式、九年级指数运算会继续深化负指数。

- Q: a⁵÷a² 等于什么？ ｜ A: a³。
