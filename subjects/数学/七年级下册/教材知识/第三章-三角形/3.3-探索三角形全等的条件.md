# 探索三角形全等的条件

## 概念 / 定义
判断两个三角形全等，不必量所有边和角——只要满足一定**组合条件**就够了。记住五个字：**边边边、边角边、角边角、角角边**（直角三角形还有斜边直角边 HL）。

## 公式 · 规则 · 结论（要能直接背）
- **SSS**：三边对应相等 → 全等。
- **SAS**：两边及其**夹角**对应相等 → 全等（角必须是两边的夹角！）。
- **ASA**：两角及其**夹边**对应相等 → 全等。
- **AAS**：两角及其中一角的对边对应相等 → 全等。
- **HL**（仅直角三角形）：斜边和一条直角边对应相等 → 全等。
- **不能判定**：SSA（边边角）——两边一对角一般**不能**判定全等（可能两个不同三角形）。

## 典型例题（带完整步骤）
> 例：如图，AB = DE，BC = EF，AC = DF，说明 △ABC ≌ △DEF。
>
> 解：
> 在 △ABC 与 △DEF 中，
> ∵ AB = DE，BC = EF，AC = DF，
> ∴ △ABC ≌ △DEF **（SSS）**。
>
> 例 2：AB = AD，∠BAC = ∠DAC，AC 公共，证 △ABC ≌ △ADC。
>
> 解：AB = AD，∠BAC = ∠DAC（夹角），AC = AC（公共边），
> ∴ △ABC ≌ △ADC **（SAS）**。

## ⚠️ 易错点 · 陷阱
- **SAS 的角必须是夹角**：两边对角（SSA）不能证全等。
- 写证明要列**三个对应相等**，再写判定依据缩写。
- **公共边、公共角、对顶角**常作为隐藏条件。
- HL 只用于**直角三角形**，且必须是斜边+直角边。

## 变式 · 拓展 · 中考链接
- 中考证明题核心技能；常与平行线、角平分线、等腰三角形结合。
- 作图（3.4）和测距（3.5）都依赖全等判定。

## 一问一答（用于引导式提问，AI 复习时优先用这些来问她）
- Q: 三边对应相等用什么判定？ ｜ A: SSS。
- Q: SAS 里的角必须是哪种角？ ｜ A: 两边的夹角。
- Q: SSA 能判定全等吗？ ｜ A: 一般不能。
