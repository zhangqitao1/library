# 两条直线的位置关系

## 概念 / 定义
同一平面内，两条直线要么**相交**（有一个交点），要么**平行**（永不相交）。相交时形成四个角；其中**对顶角相等**，**邻补角互补**（和 180°）。垂直是相交的特殊情况——交角为 90°。

## 公式 · 规则 · 结论（要能直接背）
- **对顶角相等**：∠1 与 ∠3 是对顶角 → ∠1 = ∠3。
- **邻补角互补**：∠1 + ∠2 = 180°。
- **垂直**：a ⊥ b 表示所成角为 90°；**垂线段最短**——点到直线的距离 = 该点到直线的垂线段长度。
- **同位角、内错角、同旁内角**：两条直线被第三条直线所截时形成，为后面判平行做准备。
- **经过一点有且只有一条直线与已知直线垂直**（在同一平面内）。

## 典型例题（带完整步骤）
> 例 1：如图，直线 AB、CD 相交于 O，若 ∠AOC = 50°，求 ∠BOD、∠AOD。
>
> 解：
> 第 1 步：∠AOC 与 ∠BOD 是对顶角 → ∠BOD = **50°**。
> 第 2 步：∠AOD 与 ∠AOC 邻补 → ∠AOD = 180° − 50° = **130°**。
>
> 例 2：点 P 到直线 l 的距离是 3 cm，说明什么？
>
> 解：P 到 l 的**垂线段**长为 3 cm；P 到 l 上任意其他点的线段都 ≥ 3 cm（垂线段最短）。

## ⚠️ 易错点 · 陷阱
- **对顶角必须"相对"**，不是相邻的角。
- **距离是垂线段的长度**，不是任意连线的长度。
- 说"垂直"要指明**哪两条线**垂直。
- 同位角/内错角相等**只在平行时成立**，相交时一般不相等。

## 变式 · 拓展 · 中考链接
- 角度计算题常与平行线性质、三角形内角和综合。
- 尺规作垂线、作图题中考常考。
