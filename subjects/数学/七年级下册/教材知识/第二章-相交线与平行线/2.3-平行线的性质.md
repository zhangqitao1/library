# 平行线的性质

## 概念 / 定义
已经知道两条线平行了，能推出什么角的关系？**同位角相等、内错角相等、同旁内角互补**——这是"由平行推角"的性质，和判定定理是反过来的逻辑。

## 公式 · 规则 · 结论（要能直接背）
- **性质定理**（已知平行，求角）：
  1. 两直线平行 → **同位角相等**。
  2. 两直线平行 → **内错角相等**。
  3. 两直线平行 → **同旁内角互补**。
- **平行传递性**：若 a ∥ b，b ∥ c，则 a ∥ c。
- **平行于同一条直线的两条直线互相平行**。
- 解题套路：先证平行（判定）→ 再用性质求角；或已知平行直接用性质。

## 典型例题（带完整步骤）
> 例：已知 a ∥ b，直线 c 截 a、b，∠1 = 65°（∠1 为 a 与 c 的同位角），求 ∠2（∠2 为 b 与 c 的内错角，与 ∠1 对应）。
>
> 解：
> 第 1 步：∵ a ∥ b，∴ 同位角 ∠1 等于 b 上的同位角，设为 ∠1′，∠1′ = 65°。
> 第 2 步：∠2 与 ∠1′ 是内错角，a ∥ b 时内错角相等，∠2 = **65°**。
> （也可一步：同位角相等直接得 b 上同位角 65°，内错角也 65°。）

## ⚠️ 易错点 · 陷阱
- **性质的前提是两线已平行**，不能无平行硬用。
- 求角时画**截线示意图**，标清同位/内错/同旁内角。
- 别把"对顶角相等"和"内错角相等"混用场景。
- 多步证明：证平行用判定，求角用性质，步骤分开写。

## 变式 · 拓展 · 中考链接
- 与三角形内角和、平角、对顶角组合是经典综合题。
- 八年级平行四边形、九年级相似都建立在平行线性质上。
