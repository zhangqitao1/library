二、核心公式 / 定理 / 法则速查（合书要能默写）

板块公式 / 定理（全用 Unicode）
同底数幂乘除aᵐ·aⁿ = aᵐ⁺ⁿ（底不变指数相加）；aᵐ÷aⁿ = aᵐ⁻ⁿ（a≠0，指数相减）
幂的乘方·积的乘方(aᵐ)ⁿ = aᵐⁿ（指数相乘）；(ab)ⁿ = aⁿbⁿ（每个因式都要乘方）
零指数·负指数a⁰ = 1（a≠0）；a⁻ⁿ = 1/aⁿ（a≠0）
平方差公式(a+b)(a−b) = a²−b²（"和 × 差 = 平方差"）
完全平方公式(a±b)² = a² ± 2ab + b²（首² ± 2×首×尾 + 尾²，中间别漏 2ab）
整体代换常用a²+b² = (a+b)²−2ab = (a−b)²+2ab
相交线对顶角相等；邻补角互补（和 = 180°）；点到直线的距离 = 垂线段长
平行 判定同位角相等 / 内错角相等 / 同旁内角互补（和180°）⇒ 两直线平行
平行 性质两直线平行 ⇒ 同位角相等 / 内错角相等 / 同旁内角互补
三角形两边之和 > 第三边，两边之差 < 第三边；内角和 = 180°；外角 = 不相邻两内角之和
全等判定SSS、SAS（夹角）、ASA、AAS、HL（仅直角△：斜边+直角边）；SSA 一般不行
轴对称对称点连线被对称轴垂直平分；等腰△两底角相等、三线合一；等边△三角都 60°
对称轴条数长方形 2 条、正方形 4 条、等边三角形 3 条、圆无数条
概率等可能：P = m/n（有利数÷总数）；0 ≤ P ≤ 1；必然 P=1，不可能 P=0；所有结果概率之和 = 1

三、高频易错点（考前最后扫一遍）

易错处正确做法
同底数幂乘 vs 幂的乘方a³·a² = a⁵（相加）；(a³)² = a⁶（相乘），别混
积的乘方漏系数(2xy)² = 4x²y²，(−2a²b)³ = −8a⁶b³，系数也要乘方
完全平方头号坑(a+b)² ≠ a²+b²，中间必须有 ±2ab
平方差 vs 完全平方(a+b)(a−b) 才是平方差；(a+b)(a+b)=(a+b)² 是完全平方
判定 vs 性质方向角相等 → 平行（判定）；平行 → 角相等（性质），别用反
同旁内角是互补（和180°），不是相等
SAS 的角必须是两边的夹角；SSA（边边角）一般不能证全等
等腰三角形求角给"一个内角"未说顶角/底角时要分类讨论
读 s-t 图先看清横纵轴是谁；水平段 = 路程不变（休息）；速度看增量比
概率分母必须先确认"等可能"，列举不重不漏，答案化最简分数

四、常见题型解题套路（背下来直接套）

  • 整式化简求值 → 三步：① 用乘法公式 / 去括号展开 → ② 合并同类项化到最简 → ③ 最后才代入数值（负数代入要加括号）。
  • 整体代换 → 凑公式：见到"已知 a+b、ab 求 a²+b²"，套 a²+b² = (a+b)²−2ab，把已知整体代进去，别去解 a、b。
  • 平行线几何证明 → 书写格式：写"∵（已知/已证的角相等条件）→ ∴ 两直线平行（写清依据：同位角相等/内错角相等/同旁内角互补）"。每一步都要有依据，不能跳。两平行线间有拐点时，过拐点作平行线（辅助线），把大角拆成两个内错角。
  • 全等证明 → 三对 + 依据：先找齐 3 个对应相等条件（公共边、对顶角、平行得到的角是隐藏条件），按"在△…与△…中，∵…∴≌（SSS/SAS/ASA/AAS/HL）"书写；要对应边 / 对应角时先证全等再用。
  • 变量应用题 → 三步：① 定自变量、因变量 → ② 因变量写左边、用自变量列关系式 → ③ 代入或读图作答（注意取值范围，如时间≥0、边长>0）。
  • 概率题 → 三步：① 列出所有等可能结果（不重不漏）→ ② 数有利结果数 m → ③ P = m/n 化最简分数；必然写 1、不可能写 0。

一、选择题（每题3分，共30分）
1. 下列计算正确的是 [考点：幂的运算法则（难度：基础）]
A. a²·a³ = a⁶
B. (a²)³ = a⁵
C. a⁶÷a² = a⁴
D. (2a)² = 2a²
答案与解析
C
同底数幂相除指数相减：a⁶÷a² = a⁴，C 对。A 应是 a⁵（相加）；B 应是 a⁶（相乘）；D 应是 4a²（系数也要平方）。
2. 计算 (x+2)² 的结果是 [考点：完全平方公式（难度：基础）]
A. x²+4
B. x²+2x+4
C. x²+4x+4
D. x²+4x+2
答案与解析
C
(a+b)² = a²+2ab+b²，这里 a=x、b=2：x² + 2·x·2 + 2² = x²+4x+4。漏掉中间 4x 就会误选 A。
3. 2⁻³ 的值是 [考点：负指数幂（难度：基础）]
A. 1/8
B. −8
C. −6
D. −1/8
答案与解析
A
负指数变倒数：2⁻³ = 1/2³ = 1/8。负指数不会让结果变负数。
4. 直线 AB 与 CD 被第三条直线所截，AB∥CD。若一对同旁内角中的 ∠1 = 110°，则与它同旁的内角 ∠2 = [考点：平行线性质·同旁内角（难度：基础）]
A. 70°
B. 110°
C. 20°
D. 90°
答案与解析
A
两直线平行，同旁内角互补（和 = 180°）：∠2 = 180° − 110° = 70°。注意同旁内角是"互补"不是"相等"。
5. 下列各组线段（单位：cm）能组成三角形的是 [考点：三角形三边关系（难度：基础）]
A. 1, 2, 3
B. 3, 4, 8
C. 3, 4, 5
D. 2, 2, 5
答案与解析
C
判定：最短两边之和要大于第三边。C 中 3+4=7 > 5，可以。A 中 1+2=3 不大于 3；B 中 3+4=7 < 8；D 中 2+2=4 < 5，都不行。
6. 三角形的一个外角等于 120°，与它不相邻的一个内角是 50°，则与它不相邻的另一个内角是 [考点：三角形外角性质（难度：中档）]
A. 60°
B. 70°
C. 50°
D. 120°
答案与解析
B
外角 = 与它不相邻的两个内角之和：120° = 50° + 另一个内角，所以另一个内角 = 120° − 50° = 70°。
7. 两个三角形满足"两条边及它们的夹角对应相等"，判定它们全等的依据是 [考点：全等判定（难度：中档）]
A. SSS
B. SAS
C. ASA
D. SSA
答案与解析
B
两边及"夹角"对应相等是 SAS。注意角必须是两边的夹角；SSA（两边及其中一边的对角）一般不能判定全等。
8. 下列图形中，对称轴条数最多的是 [考点：轴对称·对称轴（难度：基础）]
A. 长方形
B. 正方形
C. 等边三角形
D. 等腰三角形
答案与解析
B
长方形 2 条、正方形 4 条、等边三角形 3 条、等腰三角形 1 条，正方形最多。
9. 掷一枚均匀骰子一次，朝上的点数大于 4 的概率是 [考点：等可能事件概率（难度：基础）]
A. 1/2
B. 1/3
C. 1/6
D. 2/3
答案与解析
B
共 6 种等可能结果，大于 4 的是 5、6 共 2 种，P = 2/6 = 1/3。
10. 一根弹簧原长 10 cm，在弹性限度内每挂 1 kg 重物伸长 0.5 cm。挂 x kg 时弹簧长 y cm，则 y 与 x 的关系式为 y = 10 + 0.5x。当挂 6 kg 时弹簧长为 [考点：用关系式表示变量关系（难度：基础）]
A. 13 cm
B. 16 cm
C. 3 cm
D. 10 cm
答案与解析
A
把 x = 6 代入：y = 10 + 0.5×6 = 10 + 3 = 13（cm）。这里 x 是自变量、y 是因变量。
二、填空题（每题3分，共24分）
11. 计算：(−2a²b)³ = ______ [考点：积的乘方（难度：基础）]
答案与解析
−8a⁶b³
每个因式都要乘方：(−2)³ = −8，(a²)³ = a⁶，b³ = b³，合起来 −8a⁶b³。系数的符号和乘方都别漏。
12. 计算：(3a+2b)(3a−2b) = ______ [考点：平方差公式（难度：基础）]
答案与解析
9a²−4b²
"和 × 差 = 平方差"：(3a)² − (2b)² = 9a² − 4b²。
13. 若 x² + kx + 9 是一个完全平方式，则 k = ______ [考点：完全平方式（难度：中档）]
答案与解析
±6
完全平方式形如 (x±3)²，因为尾项 9 = 3²。中间项 2×x×3 = 6x，所以 k = +6（对应 (x+3)²）或 k = −6（对应 (x−3)²）。两个解都要写，别漏负的。
14. 已知 a + b = 5，ab = 6，则 a² + b² = ______ [考点：完全平方·整体代换（难度：中档）]
答案与解析
13
用 a²+b² = (a+b)² − 2ab = 5² − 2×6 = 25 − 12 = 13。整体代换，不需要去求 a、b 各是多少。
15. 在 △ABC 中，∠A = 70°，∠B = 60°，则 ∠C = ______ [考点：三角形内角和（难度：基础）]
答案与解析
50°
三角形内角和 180°：∠C = 180° − 70° − 60° = 50°。
16. 等腰三角形的一个内角为 80°，则它的顶角为 ______ [考点：等腰三角形·分类讨论（难度：中档）]
答案与解析
80° 或 20°
题目没说 80° 是顶角还是底角，要分两种：① 80° 是顶角，则顶角 = 80°；② 80° 是底角，则两底角共 160°，顶角 = 180° − 160° = 20°。漏掉一种就扣分。
17. 一个不透明的袋中装有 3 个红球和 5 个白球，它们除颜色外都相同。从中随机摸出一个球，摸到白球的概率是 ______ [考点：等可能事件概率（难度：基础）]
答案与解析
5/8
共 3+5 = 8 个球，每个被摸到机会相同，白球 5 个，P(白) = 5/8（已是最简分数）。
18. 一个水箱中有水 20 L，打开放水阀后每分钟流出 2 L。设放水 t 分钟后水箱剩水 Q L，则 Q 与 t 的关系式为 ______；水箱里的水全部放完需要 ______ 分钟。 [考点：用关系式表示变量关系（难度：中档）]
答案与解析
Q = 20 − 2t；10
剩水 = 原有 − 放掉，因变量 Q 写左边：Q = 20 − 2t（0 ≤ t ≤ 10）。放完即 Q = 0：20 − 2t = 0，解得 t = 10（分钟）。
三、解答题（要写出过程，共46分）
19.（8分）计算：
（1）(−3x²y)·(2xy³)；
（2）(x+3)(x−5) − x(x−2)。 [考点：整式乘法·混合运算（难度：基础）]
答案与解析
（1）−6x³y⁴；（2）−15
（1）系数相乘 (−3)×2 = −6，同底数幂相乘 x²·x = x³、y·y³ = y⁴，得 −6x³y⁴。
（2）先各自展开：(x+3)(x−5) = x²−5x+3x−15 = x²−2x−15；x(x−2) = x²−2x。再相减：(x²−2x−15) − (x²−2x) = x²−2x−15 − x²+2x = −15。注意减号要变后一括号每一项的符号。
20.（8分）先化简，再求值：(a+2)² + (a+2)(a−2)，其中 a = −1。 [考点：乘法公式·化简求值（难度：基础）]
答案与解析
化简得 2a² + 4a；当 a = −1 时，值为 −2
(a+2)² = a²+4a+4；(a+2)(a−2) = a²−4。相加：a²+4a+4 + a²−4 = 2a²+4a。代入 a = −1：2×(−1)² + 4×(−1) = 2×1 − 4 = 2 − 4 = −2。先化简到最简再代入，又快又少错。
21.（8分）（图形用文字描述如下）两条水平直线 AB、CD，且 AB∥CD，AB 在上、CD 在下。点 P 在 AB 与 CD 之间，连接 PA、PC（A 在上方直线上，C 在下方直线上）。已知 PA 与 AB 所夹的角 ∠PAB = 40°，PC 与 CD 所夹的角 ∠PCD = 30°，且 P 落在使两角"一上一下相加"的位置。求 ∠APC 的度数。 [考点：平行线性质·拐点作辅助线（难度：压轴）]
答案与解析
∠APC = 70°
过点 P 作 PE∥AB（辅助线）。
∵ PE∥AB，∴ ∠APE = ∠PAB = 40°（两直线平行，内错角相等）。
∵ AB∥CD，PE∥AB，∴ PE∥CD，∴ ∠CPE = ∠PCD = 30°（内错角相等）。
∴ ∠APC = ∠APE + ∠CPE = 40° + 30° = 70°。
口诀：两平行线间有拐点，过拐点作平行线，大角拆成两个内错角相加。
22.（8分）（图形用文字描述如下）点 B、E、C、F 依次在同一条直线上（顺序为 B、E、C、F）。已知 AB = DE，∠ABC = ∠DEF，BE = CF。求证：△ABC ≌ △DEF。 [考点：全等判定·隐藏条件（难度：压轴）]
答案与解析
△ABC ≌ △DEF（SAS）
先求第三个条件 BC = EF：
∵ B、E、C、F 在一条直线上，∴ BC = BE + EC，EF = EC + CF。
∵ BE = CF（已知），∴ BE + EC = CF + EC，即 BC = EF。
在 △ABC 与 △DEF 中：
AB = DE（已知），∠ABC = ∠DEF（已知），BC = EF（已证），
∴ △ABC ≌ △DEF（SAS）。
关键：BE = CF 两边同加 EC 凑出 BC = EF，这是隐藏条件。
23.（8分）小明骑车去图书馆，他的"路程 s（km）随时间 t（min）变化"的图象是一条折线，分三段：第 1 段 0~20 min 匀速骑到距家 3 km 的同学家；第 2 段 20~30 min 在同学家停留；第 3 段 30~45 min 匀速骑到距家 6 km 的图书馆。
（1）这个问题中，自变量和因变量各是什么？
（2）第 1 段中小明的骑车速度是多少（km/h）？
（3）20~30 min 这段时间，路程 s 有没有变化？说明小明在做什么？
（4）第 3 段中小明的骑车速度是多少（km/h）？ [考点：读 s-t 图象·分段求速度（难度：中档）]
答案与解析
（1）自变量：时间 t；因变量：路程 s。（2）9 km/h。（3）路程不变，小明在停留（休息）。（4）12 km/h。
（2）第 1 段路程增 3 km，时间 20 min = 1/3 h，速度 = 3 ÷ (1/3) = 9（km/h）。
（3）图象在这段是水平的，s 不变，说明人没有前进，在同学家停留。
（4）第 3 段从 3 km 到 6 km，增 3 km，时间 30→45 共 15 min = 1/4 h，速度 = 3 ÷ (1/4) = 12（km/h）。速度要用"路程增量 ÷ 时间增量"，别用坐标乱除。
24.（6分）一个可以自由转动的转盘被平均分成 6 个相同的扇形，依次标上数字 1、2、3、4、5、6。转动转盘，待其停止后指针指向某个扇形（指针指向边界忽略不计，每个扇形机会相同）。求：
（1）指针指向的数字是 3 的倍数的概率；
（2）指针指向的数字是质数的概率；
（3）指针指向的数字大于 6 的概率。 [考点：等可能事件·必然与不可能（难度：中档）]
答案与解析
（1）1/3；（2）1/2；（3）0
总共 6 种等可能结果。
（1）3 的倍数有 3、6 共 2 个，P = 2/6 = 1/3。
（2）质数有 2、3、5 共 3 个（1 不是质数，4、6 是合数），P = 3/6 = 1/2。
（3）1~6 没有大于 6 的，是不可能事件，P = 0。
25.（10分）已知 a − b = 3，ab = 4。求：
（1）a² + b² 的值；
（2）(a + b)² 的值。 [考点：完全平方公式·整体代换（难度：压轴）]
答案与解析
（1）a² + b² = 17；（2）(a + b)² = 25
（1）由 (a−b)² = a² − 2ab + b² 得 a² + b² = (a−b)² + 2ab = 3² + 2×4 = 9 + 8 = 17。
（2）(a+b)² = a² + 2ab + b² = (a²+b²) + 2ab = 17 + 2×4 = 17 + 8 = 25。
两小题都靠"a²+b² 当整体"来回代换，不用真的解出 a、b。
