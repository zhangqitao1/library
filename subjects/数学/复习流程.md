# 数学 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**数学专属打法**。
> 数学 = 强连贯型，复习核心是"**懂概念本质 + 规范运算 + 错题归零**"，不是盲目刷题。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。数学常需回溯前册（有理数→整式→方程），发现断点要回补。

## 二、考什么（四大维度）
1. **概念/定理本质**：能说清"为什么"，不只会套公式。
2. **规范运算**：步骤完整、符号不丢、移项变号。
3. **典型应用题**：审题→列式→解→检验。
4. **几何**：图形性质 + 证明思路 + 辅助线。

## 三、数学专属方法（怎么高效）
- **一题三遍 > 三题一遍**：一道典型题做透,胜过刷一堆。
- **讲题（费曼）讲思路**：让她**讲**这题"先想到啥、为什么这么做",讲不顺=没真懂。
- **限时计算训练**：每天一组限时口算/计算，专治"会做但算错"。
- **错题先独立重做再对答案**：做对了追问"和上次错的思路差在哪"；仍错则精准定位卡在哪一步。
- **概念别死背**：用例子/反例让她自己说出定义边界。

## 四、高频坑（优先攻 + 错题四类归因）
做错先归因，对症下药：
- **计算粗心型**：符号、移项、去括号 → 限时计算 + 慢核对。
- **概念不清型**：定义/定理用错 → 回看知识卡，举例辨析。
- **方法不会型**：没思路 → 讲解 + 同类变式 1~2 道。
- **审题失误型**：应用题看错条件 → 圈关键词、画图。

## 五、🎯 命题新方向落点（2026 · 据济南中考新政）

> 2026 起济南名校"综合评价招生"看重思维与应用，命题随之转向。下面把 4 个方向翻成**数学可操作的练法**——复习时按这些来练，别再只刷套路计算。配比与情境化原则见 `AGENTS.md` 九.2 与 `exam-maker` SKILL。

1. **减记忆、增高阶（少"算什么"，多"为什么/怎么用/比一比"）**
   - 公式不止会套，要能说**为什么**：让她讲 "(a+b)²为什么中间有 2ab"（画正方形拼图）、"SSA 为什么不能判全等"（举反例）。
   - 多问**比较类**："平行线判定和性质差在哪""SAS 和 SSA 差一个什么字"。
   - 练法：每个公式/定理配 1 道"辨析题"（判断对错并说理），而非只配计算题。

2. **长情境先提取（真实长材料 → 先读出关键信息 → 再建模解题）**
   - 选**交通/消费/本土文化**背景：公交/地铁里程票价、超市满减、共享单车计费、济南泉水流量/趵突泉门票、大明湖游船。
   - 练"两步走"：① 先圈出题里**哪个量在变、哪个跟着变**（找自变量/因变量）、关键数据；② 再列关系式或读图。
   - 练法：给一段 4~6 行的生活材料，先只问"自变量是谁、关系式怎么列"，再问求值——专治"会算但读不懂题"。

3. **跨学科 + 动态几何（旋转/折叠/隐圆 + 跨模块综合 + 压轴层层递进）**
   - **折叠（翻折）**：把"折叠后重合的边角相等"当成隐藏的全等/轴对称条件——折痕是对称轴，对应边角相等。练：长方形折一角求角度/长度。
   - **旋转**：旋转前后**对应线段相等、对应角相等、旋转角相等**；练"绕某点转 60°/90° 后求新坐标或证全等"。
   - **跨模块**：几何里嵌变量关系（动点走，路程/面积随时间变 → 列关系式/读分段图）；整式运算包进几何周长面积。
   - **压轴递进设问**：一道大题分 (1)(2)(3)，从"求一个具体值"→"用字母表示"→"探究/分类讨论"，一步一台阶。练时先把 (1) 拿稳，再往上爬。

4. **重过程规范（解答题逐步写、过程完整、化简到底）—— 名校在这拉分**
   - **证明三段式**：先写"在△…和△…中"列三个对应相等条件，再写判定依据（SSS/SAS/ASA/AAS/HL），最后"∴"结论；公共边/对顶角等**隐藏条件要写出来**。
   - **计算分步**：先写用哪个公式/法则，再代入，再化简，**不跳步**；完全平方别漏 2ab、去括号变号要写出来。
   - **化简到底**：结果要最简（合并同类项、分数化最简、按字母降幂排）。
   - 练法：做完对答案时，不只看结果对不对，**对照解析逐步检查"步骤写全没"**——这是按名校评分标准自查。

## 六、期末三阶段节奏（对标历下）
1. **回归课本**：把概念/公式弄"懂"（合书复述定义+推导），先不刷题。
2. **专项突破**：选 2~3 个最弱专题（如乘法公式化简、几何全等证明、动态几何折叠/旋转）集中攻，每题做透+变式。
3. **限时模拟**：限时整卷，练时间分配；先易后难，压轴题先把 (1)(2) 拿稳、写出公式拿步骤分。

## 七、当堂自测形式（交给 `exam-maker`）
限时计算组、概念辨析（说理判断）、长情境建模题、动态几何（折叠/旋转）、几何证明（重过程规范）、错题变式题（换数字/换情境）。

## 八、闭环沉淀
错题入 `错题本/`（标四类归因）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
