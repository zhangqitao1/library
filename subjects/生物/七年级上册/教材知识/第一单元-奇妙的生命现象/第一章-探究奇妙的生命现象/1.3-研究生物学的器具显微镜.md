# 研究生物学的器具——显微镜

## 概念 / 定义
**光学显微镜**是看细胞这类"小到肉眼看不见"的东西的放大工具。它靠两组镜片（**目镜 + 物镜**）一起把物体放大。这一节**识图题、操作题年年考**，结构名称和"动哪个、怎么动"必须背熟。

## 必背结构（识图必背，按"看-放-照-调"记）
| 部位 | 名称 | 作用 |
|---|---|---|
| 看的镜片 | **目镜**（插在镜筒上，无螺纹） | 靠眼睛看，放大物像 |
| | **物镜**（装在转换器上，有螺纹） | 靠近标本，放大物像 |
| 放标本 | **载物台**（中间有通光孔，两侧压片夹） | 放玻片标本 |
| 照明 | **遮光器**（有大小光圈） | 调进光量大小 |
| | **反光镜**（一面平、一面凹） | 反射光线照亮标本，凹面镜聚光更强 |
| 调焦 | **粗准焦螺旋** | 大幅升降镜筒，**快速找到物像** |
| | **细准焦螺旋** | 小幅升降，**把物像调清晰** |
| 连接转换 | **转换器** | 转动它来更换物镜 |
| 支撑 | **镜座、镜柱、镜臂、镜筒** | 支撑与连接 |

## 必背结论（操作与计算，要能直接背）
**使用步骤：取镜安放 → 对光 → 放片（安放标本）→ 观察 → 整理收镜。**
1. **对光**：转动转换器使**低倍物镜**对准通光孔 → 选**大光圈** → 用反光镜（光暗用**凹面镜**）调到看见**明亮的白色圆形视野**。
2. **观察**：标本放载物台、压片夹压住，标本对准通光孔；先**从侧面看着物镜**转粗准焦螺旋让镜筒**下降**到接近标本（防压碎玻片）；再**眼睛看目镜**、反向转粗准焦螺旋让镜筒**慢慢上升**直到看到物像；最后用**细准焦螺旋**调清晰。

**核心规律（必背）：**
- **放大倍数 = 目镜倍数 × 物镜倍数**。
- 放大倍数越**大** → 看到的细胞**越大越少**、视野**越暗**；倍数越**小** → 细胞**越小越多**、视野**越亮**。
- **物像是倒立的**：上下左右都相反。所以**物像偏哪个方向，玻片就往哪个方向移**（"偏哪移哪"，同向移动玻片）。
- 物镜：**越长放大倍数越大、离标本越近**；目镜：**越短放大倍数越大**。

## 典型例题（识图 + 操作）
> 例1（移动方向）：视野中物像在**左上方**，要把它移到中央，玻片应往哪移？
> 解：物像是倒像，"偏哪移哪"——物像在左上方，就把玻片往**左上方**移。（口诀：往同一方向移玻片）答：左上方。
>
> 例2（计算+判断）：目镜10×、物镜10× 时视野细胞较多；换成物镜40× 后，视野会怎样变化？
> 解：放大倍数从 10×10=100 倍 → 10×40=400 倍。倍数变大 → 细胞**变大、变少**，视野**变暗**，需换**大光圈/凹面镜**补光。
>
> 例3（找物像找不到）：转动粗准焦螺旋没找到物像，最可能哪步错了？
> 解：很可能下降镜筒时没从侧面看，或没把标本对准通光孔/没对好光。规范做法：先侧看物镜降镜筒，再看目镜升镜筒找像。

## ⚠️ 易错点 · 陷阱
- **物像是倒立的**：写字"上"在视野里看到的是"下"翻转的样子；移动方向**与物像同向**移玻片（很多人记反成"反向"）。
- **粗 / 细准焦螺旋分工**：粗的"找像"、细的"调清晰"，别答反。
- **低倍换高倍**：先在低倍找到并移到中央，再换高倍；换高倍后**只能用细准焦螺旋**调焦（不能再大幅转粗的，会压坏玻片）。
- **视野亮暗调节**：光暗用**大光圈 + 凹面镜**；光太强用**小光圈 + 平面镜**。
- **放大的是长度/宽度，不是面积或体积**："放大100倍"指线度放大100倍。
- 镜筒下降时**必须眼睛从侧面看物镜**，不能边看目镜边下降（会把玻片压碎）。

## 一问一答
- Q: 放大倍数怎么算？ ｜ A: 目镜倍数 × 物镜倍数。
- Q: 物像在视野右下方，玻片往哪移到中央？ ｜ A: 往右下方移（偏哪移哪，同向）。
- Q: 视野太暗怎么调亮？ ｜ A: 用大光圈、凹面反光镜。
- Q: 粗准焦螺旋和细准焦螺旋各管什么？ ｜ A: 粗的快速找物像（大幅升降），细的把物像调清晰（小幅）。
- Q: 放大倍数越大，视野里细胞数目和亮度怎么变？ ｜ A: 细胞变大变少、视野变暗。
- Q: 下降镜筒时眼睛应该看哪里？为什么？ ｜ A: 从侧面看物镜，防止物镜压碎玻片。
