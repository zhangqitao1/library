# 芽是未发育的枝条或花

## 概念 / 定义
**芽**是未发育的枝条或花，是植物**生长和生殖**的"预备队"。按着生位置分顶芽、侧芽；按发育结果分枝芽、花芽、混合芽。

## 必背结论（要能直接背）
**芽的类型：**
| 分类标准 | 类型 | 说明 |
|---|---|---|
| 着生位置 | **顶芽** | 着生在茎顶端 |
| | **侧芽** | 着生在茎侧面（叶腋处） |
| 发育结果 | **枝芽** | 发育成**枝条**（茎、叶） |
| | **花芽** | 发育成**花** |
| | **混合芽** | 发育成**枝条和花**（如苹果、梨） |

**枝芽发育成枝条的顺序：**
芽轴 → **茎**；芽原基 → **叶**；芽原基发育成侧芽

**顶端优势：**
- 顶芽生长旺盛，**抑制**侧芽生长
- 摘除顶芽（打顶）→ 侧芽迅速发育 → 使植物**多分枝**（棉花打顶、果树修剪）

## 读图 · 识图示范
> **枝芽纵剖图识图**：
> - 最顶端分生组织 → 使芽**伸长**
> - 侧面层层包裹 → **叶原基**（将来发育成叶）
> - 芽轴 → 将来发育成**茎**
> - 基部小突起 → **芽原基**（将来发育成**侧芽**）
>
> **花芽**通常比枝芽**肥大**（内部已分化花器官）。

## 典型例题（带完整步骤）
> 例1：园林中把树木下部侧枝剪掉、保留顶端，目的是？
> 解：利用**顶端优势**，让树木**长得高直**（侧芽被抑制）。
>
> 例2：马铃薯块茎上的"芽眼"是什么？
> 解：**芽**（混合芽），可发育成新的植株——属于**营养繁殖**。

## ⚠️ 易错点 · 陷阱
- **顶芽≠枝芽**：顶芽是位置，枝芽是发育结果（顶芽可以是枝芽或花芽）。
- 花芽发育成**花→果实**，不是直接变果实。
- 顶端优势原理用于解释"打顶摘心"，别和"修剪侧枝"搞反目的。

## 变式 · 拓展 · 会考链接
- 与无性生殖联系：扦插利用的芽发育成新枝。
- 果树修剪：疏花疏果、打顶促侧枝提高产量。

## 一问一答
- Q: 枝芽发育成什么？ ｜ A: 枝条（茎和叶）。
- Q: 什么是顶端优势？ ｜ A: 顶芽优先生长，抑制侧芽生长。
- Q: 棉花打顶的目的是？ ｜ A: 打破顶端优势，促进侧枝发育，提高产量。
