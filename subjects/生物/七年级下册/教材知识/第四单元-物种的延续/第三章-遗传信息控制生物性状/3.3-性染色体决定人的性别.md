# 性染色体决定人的性别

## 概念 / 定义
人的性别由**性染色体**决定。女性体细胞是 **XX**，男性是 **XY**。生男生女取决于**父亲**提供的精子类型——不是母亲"决定"的，更不是迷信能改变的。

## 必背结论（要能直接背）
**人的染色体：**
- 体细胞 **23 对（46 条）**染色体
  - **22 对常染色体**（男女相同）
  - **1 对性染色体**：女性 **XX**，男性 **XY**

**性别决定机制：**
| 亲代 | 性染色体 | 产生的配子 |
|---|---|---|
| 母亲 | XX | 只有一种：**X** |
| 父亲 | XY | 两种：**X** 或 **Y**（约各一半） |

- 精子 **X** + 卵细胞 **X** → 受精卵 **XX** → **女孩**
- 精子 **Y** + 卵细胞 **X** → 受精卵 **XY** → **男孩**
- 生男生女概率各约 **50%**，与已经生过几个孩子**无关**

## 读图 · 识图示范
> **性别决定示意图**：
> ```
> 母亲 XX → 卵细胞 X
> 父亲 XY → 精子 X 或 Y
> 组合：XX（女）或 XY（男），各 50%
> ```
> 识图题：标出**父亲产生两种精子**是性别比例均衡的原因。

## 典型例题（带完整步骤）
> 例1：生男孩还是女孩由谁决定？为什么？
> 解：由**父亲提供的精子**决定。母亲只能提供 X 卵细胞；精子带 X 得女，带 Y 得男。
>
> 例2：一对夫妇已连生三女，第四胎男孩概率？
> 解：每次独立，仍是 **50%**（不是"该生男了"）。

## ⚠️ 易错点 · 陷阱
- **母亲只有 X**，不能"决定"生男生女。
- 每次怀孕生男生女概率**独立**，都是约 50%。
- **Y 染色体只有男性有**；女性的性染色体是 XX。
- 别和"第一性征/第二性征"混为一谈。

## 变式 · 拓展 · 会考链接
- 与基因遗传结合：有些性状与性染色体连锁（初中简化即可）。
- 社会议题：反对性别歧视，男女平等。
