# 遗传信息改变可引起生物变异

## 概念 / 定义
**变异**是生物亲子之间以及子代个体之间的**差异性**。有的变异能**遗传**给后代，有的只是环境造成的**暂时变化**——区分这两类是会考常考点。

## 必背结论（要能直接背）
**变异分类：**
| 分类标准 | 类型 | 原因 | 例子 |
|---|---|---|---|
| 能否遗传 | **可遗传变异** | **遗传物质改变** | 基因突变、染色体变异 |
| | **不可遗传变异** | 仅**环境**引起，遗传物质未变 | 晒黑皮肤、水土肥造成的产量差 |
| 对生存影响 | **有利变异** | 适应环境 | 抗病虫害品种 |
| | **不利变异** | 不适应环境 | 白化病 |
| | **中性变异** | 无明显利弊 | 有耳垂/无耳垂 |

- **可遗传变异**是生物进化的**原材料**。
- 变异**不一定都有利**——大多数变异中性或不利，但为自然选择提供材料。

## 典型例题（带完整步骤）
> 例1：太空椒（太空育种）个大味甜，这种变异能遗传吗？
> 解：太空射线引起**遗传物质改变** → **可遗传变异**，可稳定遗传。
>
> 例2：同一品种小麦，肥沃土地比贫瘠土地产量高，属于？
> 解：**不可遗传变异**（环境差异，遗传物质相同）。

## ⚠️ 易错点 · 陷阱
- **可遗传变异必须遗传物质改变**，环境引起的产量差异**不能遗传**。
- "一母生九子，九子各不同"既有**基因重组**（有性生殖），也有变异。
- 有利/不利是**相对环境**而言：白化在暗处无害，强光下不利。

## 一问一答
- Q: 什么是可遗传变异？ ｜ A: 遗传物质发生改变，能传给后代的变异。
- Q: 晒黑皮肤能遗传吗？ ｜ A: 不能，是环境引起的不可遗传变异。
- Q: 变异对进化有什么意义？ ｜ A: 可遗传变异为自然选择提供原材料。
