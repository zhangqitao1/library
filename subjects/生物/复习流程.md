# 生物 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**生物专属打法**。
> ⚠️ **生物是八年级会考科目（学完即考、计入中考）**，从七年级就要抓扎实，别当副科。
> 命题对标济南"等级学科" **7:2:1**（基础占七成）——**把基础啃透就拿大头分**。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。

## 二、考什么（四大维度）
1. **核心概念准确**：细胞结构、光合/呼吸、生态系统组成等，定义要精准。
2. **识图 / 示意图**：结构图、过程图——看图能说出各部分名称与功能。
3. **实验**：实验目的、步骤、现象、结论。
4. **联系生活的情境应用**（命题趋势）：用知识解释生活现象。

## 二.5、🔴 命题新方向落点（2026 · 据济南中考新政）
> 济南 2026 起综合评价招生，**生物会考等级是名校硬门槛（多校要 A）**。命题已从"背知识点"转向"用素养解决真实问题"。下面把 `exam-maker` 的四个出题动作翻译成**生物可操作的练法**——平时就按这个练，考场才不慌。

1. **减记忆、增高阶**：少问"是什么"（光合场所是？），多问"为什么 / 比较 / 怎么用"。
   - 练法：每讲完一个概念，追一句"那……为什么 / 和谁像 / 用在哪"。例：不止问"呼吸场所"，而问"储粮为什么要低温干燥？"。
2. **长情境先提取**：题干给农业生产 / 生态保护 / 健康 / 育种的真实长材料，**先从材料里圈关键信息，再调知识作答**。
   - 练法：读题时先用笔划出"问的是什么、给了哪些条件/数据"，把材料语言翻译成课本概念，再答。专治"材料一长就懵"。
3. **实验探究为重（生物的命脉）**——按**探究五环节**练，缺一不可：
   - ① **提出问题/作假设** ② **变量控制**（找出单一变量，其余相同；认准对照组/实验组）③ **过程现象**（步骤顺序、为什么这么做）④ **数据处理**（看表/曲线说趋势、算差值、找拐点）⑤ **误差与改进**（哪步会带来误差、怎么让结论更可靠）。
   - 高频实验：光合验证（暗处理、酒精脱色、碘液）、种子萌发条件对照、蒸腾（凡士林/油封）、光照强度与光合。
   - 练法：每个实验都让小雨答全"为什么暗处理、为什么用酒精、变量是谁、结论凭什么得出"。
4. **重过程规范（拉分点）**：探究题要**术语规范、结论有依据**——结论句式"因为……（现象/数据），所以……（结论）"，别只写一个词。
   - 练法：批改时先看"有没有把依据写出来"；只写答案不写依据的，打回重写。

> 识图怎么练（生物特别重）：盖住图注→指着图说每部分名称+功能→再说清"过程/路线"（如子房→果实、染色体→DNA→基因层层包含）。没有真图时，把图的关键信息用文字讲清再问。

## 三、生物专属方法（怎么高效）
- **思维导图建体系**（最重要）：按"细胞→组织→器官→系统"或"生态系统结构"把零散知识连成网，治"会了这题、换情境就不会"。
- **看图说话**：盖住图注，让她指着图说每部分叫什么、干什么。
- **概念精准辨析**：相近概念成对辨（如光合作用 vs 呼吸作用、动物细胞 vs 植物细胞）。
- **输出学习法**：让她用自己的话讲清一个过程（如"食物怎么被消化吸收"）。
- **联系生活**：多问"为什么……"（为什么白天给植物浇水？），把知识用起来。

## 四、高频坑（优先攻）
- **相近概念混**（光合↔呼吸、动物细胞↔植物细胞结构）。
- **图的结构记错**（线粒体/叶绿体功能搞反）。
- 实验步骤/变量说不清。

## 五、期末/会考三阶段节奏（对标历下）
1. **回归课本**：以"基础"为主（占七成分），按目录过概念+图，合书复述。
2. **专项突破**：错题归因，专攻易混概念与识图题。
3. **限时模拟**：限时整卷，适应题型；情境题练"用知识解释现象"。

## 六、当堂自测形式（交给 `exam-maker`）
识图作答、概念辨析（选择/判断）、过程排序、**真实情境分析题**、**实验探究题（变量/数据/误差/改进）**。**难度比按 7:2:1**（基础占七成）；情境与探究题用来练"用知识解决真实问题"，但仍以基础概念为底。

## 七、闭环沉淀
错题入 `错题本/`（易混概念、错图单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
