# 英语七上 · 各单元核心词汇

> 词性缩写：n.名词 v.动词 adj.形容词 adv.副词 prep.介词 pron.代词 conj.连词 num.数词
> ⚠️待核对：以下为按 2024 人教版各单元**话题**整理的核心词，**不是逐页完整单词表**；开学请对照课本补充拼写、音标与遗漏词。

## Starter Unit 1 Hello!（问候 / 姓名）
- hello / hi *interj.* 你好
- good *adj.* 好的；morning / afternoon / evening *n.* 上午/下午/晚上
- name *n.* 名字
- fine *adj.* 健康的，很好的
- thanks *n.* 谢谢
- spell *v.* 拼写
- meet *v.* 遇见，认识

## Starter Unit 2 Keep Tidy!（整洁 / 物品 / 物主代词）
- tidy *adj.* 整洁的（untidy 不整洁的）
- room *n.* 房间
- sofa *n.* 沙发
- but *conj.* 但是
- our *pron.* 我们的
- always *adv.* 总是
- everywhere *adv.* 到处

## Starter Unit 3 Welcome!（动物 / 农场 / 指示代词）
- animal *n.* 动物；dog, cat, duck, chicken, rabbit, cow, horse, sheep, pig, goose（鹅，复数 geese）
- carrot, potato, tomato, apple, flower, tree *n.* 蔬果植物
- farm *n.* 农场；yard *n.* 院子；lake *n.* 湖
- count *v.* 数数；plant *n./v.* 植物 / 种植
- big / small *adj.* 大的 / 小的
- behind *prep.* 在……后面；at *prep.* 在（某处/某时）

## Unit 1 You and Me（自我介绍 / 交友）
- grade *n.* 年级；classmate *n.* 同班同学；class *n.* 班级
- country *n.* 国家（复数 countries）
- hobby *n.* 爱好（复数 hobbies）
- mistake *n.* 错误
- band *n.* 乐队；guitar *n.* 吉他；tennis *n.* 网球
- same *adj.* 相同的；both *pron./adj.* 两个都
- each other 互相；twin *n.* 双胞胎之一
- would *v.* （礼貌邀请）想要

## Unit 2 We're Family!（家庭成员）
- family *n.* 家庭，家人；photo *n.* 照片
- parent *n.* 父/母；grandparent *n.* （外）祖父母
- father, mother, brother, sister, uncle, aunt, cousin *n.* 家庭成员
- son *n.* 儿子；daughter *n.* 女儿；husband *n.* 丈夫；wife *n.* 妻子
- love *v.* 爱；young *adj.* 年轻的

## Unit 3 My School（学校 / 方位）
- classroom, library, playground, office, lab, building, gate, hall *n.* 学校设施
- furniture *n.* 家具（不可数）；map, picture, wall, flower, tree *n.*
- floor *n.* 楼层；first/second/third *num.* 第一/二/三
- in / on / under / behind / near / between *prep.* 方位介词
- next to 紧邻；in front of 在……前面
- some / any 一些；many / much 许多；a lot of 大量
- how many / how much 多少
- left *n./adj.* 左；right *n./adj.* 右

## Unit 4 My Favourite Subject（学科 / 喜好）
- subject *n.* 学科；math, Chinese, English, history, geography, science, music, art, P.E. 学科名
- favourite *adj.* 最喜欢的
- because *conj.* 因为
- interesting / boring / difficult / easy / fun / useful / relaxing *adj.* 描述词
- be good at 擅长
- free *adj.* 空闲的，免费的

## Unit 5 Fun Clubs（社团 / 才艺）
- club *n.* 社团；join *v.* 加入
- can *modal v.* 会，能
- sing, dance, swim, draw, speak *v.* 才艺动词
- play the guitar / piano（乐器加 the）
- play basketball / football / chess（球类棋类不加 the）

## Unit 6 A Day in the Life（日常作息 / 时间）
- get up 起床；have breakfast/lunch/dinner 吃饭
- go to school / go to bed / do homework / watch TV / brush teeth 作息
- time *n.* 时间；o'clock 点钟
- what time / when 几点/何时
- early *adv./adj.* 早；late *adv./adj.* 晚
- always / usually / often / sometimes / never *adv.* 频度副词

## Unit 7 Happy Birthday!（生日 / 日期 / 数字）
- birthday *n.* 生日；party *n.* 聚会；cake *n.* 蛋糕
- 月份：January–December（首字母大写）
- 基数词 one–twenty, twenty-one...；序数词 first, second, third, fifth, ninth, twelfth, twentieth
- date *n.* 日期；how old 多大；when 何时
- present / gift *n.* 礼物
