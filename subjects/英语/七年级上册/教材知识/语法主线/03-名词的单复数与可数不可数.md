# 名词的单复数与可数 / 不可数

## 概念 / 定义
- **可数名词**：能一个一个数的（apple, book, dog），有单数也有复数。一个的时候前面要加 **a/an**。
- **不可数名词**：不能直接数的（water, milk, furniture, music），**没有复数**，前面**不加 a/an**，永远当单数看。

## 公式 · 规则 · 结论（要能直接背）
可数名词**变复数**的五条规则：
1. 一般情况 **+s**：book → book**s**，dog → dog**s**。
2. 以 **s, x, ch, sh** 结尾 **+es**：bus → bus**es**，box → box**es**，watch → watch**es**，dish → dish**es**。
3. 以 **辅音字母 + o** 结尾 **+es**（口诀"黑人英雄爱吃土豆和西红柿"：hero, potato, tomato +es）：potato → potato**es**，tomato → tomato**es**。
   - 但 photo → photo**s**，piano → piano**s**（外来词/缩写 +s）。
4. 以 **辅音字母 + y** 结尾：变 **y 为 i 再 +es**：family → famil**ies**，hobby → hobb**ies**，country → countr**ies**。
   - 元音字母 + y 直接 +s：boy → boy**s**，key → key**s**。
5. 以 **f / fe** 结尾：变 **f/fe 为 v 再 +es**：knife → kni**ves**，leaf → lea**ves**。
- **不规则复数**（必须死记）：
  - man → **men**，woman → **women**，child → **children**，foot → **feet**，tooth → **teeth**，mouse → **mice**，person → **people**。
  - **单复数同形**：sheep → sheep，deer → deer，fish → fish（鱼通常不变），Chinese → Chinese。
- a + 辅音音素开头（a book）；an + 元音音素开头（an apple, an hour, an egg）。

## 典型例题
> 例：写出复数。① potato ② family ③ knife ④ sheep ⑤ child
>
> 解：① potato**es**（辅音+o）② famil**ies**（辅音+y）③ kni**ves**（fe→ves）④ **sheep**（同形）⑤ **children**（不规则）

## ⚠️ 易错点 · 陷阱
- ❌ two **tomatos** ✅ two **tomatoes**（辅音+o 加 es）
- ❌ many **familys** ✅ many **families**（辅音+y 变 i 加 es）
- ❌ three **sheeps** ✅ three **sheep**（单复数同形）
- ❌ a water / two waters ✅ **some water**（water 不可数，不加 a，不加 s）
- ❌ many **furnitures** ✅ **much furniture**（furniture 不可数）
- 复数主语 → be 用 are，动词不加 s：The boys **are** here. / Dogs **like** bones.

## 变式 · 拓展 · 中考链接
- 配套 some/any、many/much、how many/how much 一起考（见 06 号 there be 卡片）：
  - many / how many + **可数复数**；much / how much + **不可数**；a lot of / some / any 两者都行。
- 中考单选高频："数词 + 名词"是否一致、可数不可数搭配是否正确。
