# 连词 and / but / because

## 概念 / 定义
连词是"胶水"，把词、短语或句子连起来。七上重点三个：**and**（而且，顺承）、**but**（但是，转折）、**because**（因为，说原因）。

## 公式 · 规则 · 结论（要能直接背）
- **and**（和；而且）：连接**同类、一致**的内容。I like math **and** English.
- **but**（但是）：表示**转折、相反**。I like math, **but** I don't like history.
- **or**（或者）：表选择，多用于疑问/否定。Do you like math **or** science?
- **because**（因为）：回答 **Why** 提问，引导原因。
  - — **Why** do you like music? — **Because** it's relaxing.（因为它令人放松）
- **because vs so**：because 接**原因**，so 接**结果**，**两者不能在同一句里同时用**（这是中式英语大坑）。
  - ✅ I like it **because** it's fun. ／ It's fun, **so** I like it.
  - ❌ **Because** it's fun, **so** I like it.（because 和 so 不能连用）

## 典型例题
> 例：选 and / but / because 填空。
> ① I like P.E. ___ I don't like math.
> ② — Why do you like English? — ___ it's interesting.
>
> 解：
> ① I like P.E. **but** I don't like math.（前后相反 → 转折 but）
> ② **Because** it's interesting.（回答 Why → because）

## ⚠️ 易错点 · 陷阱
- ❌ Because it's fun, so I like it. ✅ 去掉一个：Because it's fun, I like it. 或 It's fun, so I like it.
- 回答 **Why** 的问题，要用 **Because**，不能只说 it's interesting（口语可以，书面要完整）。
- and 连接两个主语作主语时，谓语用复数：Tom **and** Jerry **are** here.

## 变式 · 拓展 · 中考链接
- 写"我喜欢某学科 + 原因"是 Unit 4 写作主线：句型 I like ... **because** it's ...（interesting / useful / fun / relaxing）。
- 中考完形高频考逻辑连词的语义判断（顺承/转折/因果）。
