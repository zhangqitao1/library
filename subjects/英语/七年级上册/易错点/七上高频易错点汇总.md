# 英语七上 · 高频易错点汇总

> 英语卡片的灵魂是"错例对照"。下面按语法主线归类，复习前先扫一遍，重点攻反复错的。

## 一、be 动词 vs 实义动词（头号大坑）
- ✗ I **am** like English. → ✓ I **like** English.（like 是动词，不用 am）
- ✗ He **is** play football. → ✓ He **plays** football. / He **is** playing...（一般现在时不用 be）
- 一句话只能有一个谓语：**有了 be 就别再加实义动词原形当谓语**。

## 二、主谓一致 / be 形式
- ✗ She **are** my friend. → ✓ She **is** my friend.（she 单数→is）
- ✗ My name **am** Gina. → ✓ My name **is** Gina.（name 单数→is）
- ✗ These **is** my books. → ✓ These **are** my books.（these 复数→are）
- ✗ Tom and Jerry **is** here. → ✓ ... **are** here.（and 连两人→复数）

## 三、第三人称单数（三单）
- ✗ He **like** music. → ✓ He **likes** music.（三单加 s）
- ✗ He **doesn't likes** it. → ✓ He **doesn't like** it.（有 doesn't→原形）
- ✗ **Does** she **likes**...? → ✓ **Does** she **like**...?（有 does→原形）
- ✗ He **haves** a dog. → ✓ He **has** a dog.（have→has）
- ✗ He **gos** to school. → ✓ He **goes**...（以 o 结尾 +es）

## 四、名词单复数 / 可数不可数
- ✗ two **tomatos** → ✓ two **tomatoes**（辅音+o 加 es）
- ✗ many **familys** → ✓ many **families**（辅音+y 变 ies）
- ✗ three **sheeps** → ✓ three **sheep**（单复数同形）
- ✗ a **water** / many **furnitures** → ✓ **some water** / **much furniture**（不可数不加 a、不加 s）
- ✗ How **much** books → ✓ How **many** books（可数复数用 many）

## 五、代词
- ✗ This is **she** book. → ✓ This is **her** book.（"她的"用 her）
- ✗ **I** name is... → ✓ **My** name is...（作定语用物主代词）
- ✗ a **my** pen / this **my** pen → ✓ **my** pen（my 前不加 a/this）
- **its（它的）≠ it's（it is）**；her 既是"她"（宾格）也是"她的"。
- 介词后用宾格：with **us**，for **me**。

## 六、指示代词 & 回答
- ✗ Is this your pen? — Yes, **this** is. → ✓ Yes, **it** is.（回答用 it）
- ✗ Are those...? — Yes, **those** are. → ✓ Yes, **they** are.（回答用 they）
- 句子变复数：This is a book → **These are books**（this→these, is→are, a 去掉, book 复数，四处同改）。

## 七、名词所有格
- ✗ my parents**'s** room → ✓ my parents**'** room（已 s 结尾只加 '）
- ✗ **children'** books → ✓ **children's** books（不以 s 结尾加 's）
- ✗ a **China's** map → ✓ a map **of China**（无生命用 of）

## 八、There be 句型
- ✗ There **has** a book. → ✓ There **is** a book.（"有"用 there be）
- ✗ There **are** a pen and two books. → ✓ There **is**...（就近原则，紧挨 a pen）
- ✗ Is there **some** water? → ✓ Is there **any** water?（疑问句用 any）

## 九、情态动词 can
- ✗ She **cans** dance. → ✓ She **can** dance.（can 不加 s）
- ✗ He can **plays** football. → ✓ He can **play** football.（can 后原形）
- ✗ Can you **to** swim? → ✓ Can you **swim**?（can 后不加 to）
- 回答 Can 问句用 can/can't，不用 do/don't。

## 十、疑问句
- ✗ What **you like**? → ✓ What **do you like**?（实义动词疑问加 do）
- ✗ Where **she is** from? → ✓ Where **is she** from?（be 提到主语前）
- ✗ Is your father a doctor? — Yes, **it is**. → ✓ Yes, **he is**.（人称一致）
- 对划线提问：先定问什么（人/物/地/时/因/数）→ 选疑问词 → 改疑问语序。

## 十一、连词 / 介词 / 大小写
- ✗ Because..., **so** ... → ✓ 只用一个（because 或 so）。
- ✗ get up **in** six → ✓ **at** six（具体时刻用 at）。
- ✗ It's **in** May 1st → ✓ It's **on** May 1st（具体某天用 on）。
- ✗ english / monday / may → ✓ **English / Monday / May**（语言、星期、月份首字母大写）。

## 十二、日期与数词
- ✗ My birthday is **1st May** → ✓ **May 1st**（人教用法：月在前 + 序数词）。
- 易错序数词：fifth, ninth, twelfth, twentieth。
- how old 问年龄，how much 问价格，别混。

## 通用方法
- **看主语定 be、定动词**：单复数 → be 形式 → 动词是否加 s/用原形，一条线判断。
- **一句一谓语**：be 和实义动词不能同时当谓语。
- **疑问/否定看谓语**：be/can 提前或加 not；实义动词借 do/does/don't/doesn't，动词回原形。
