# Unit 6 Rain or Shine

## 话题
### 天气描述
- sunny / rainy / cloudy / windy / snowy
- How's the weather?
### 季节与活动
- spring / summer / autumn / winter
- go hiking / fly kites

## 语法：时态辨析
### 一般现在时
- 规律：It rains a lot in summer.
- 标志：often / every year
### 现在进行时
- 此刻：It is raining now.
- 标志：now / Look!
### 状态动词
- know / like / want 不用进行时

## 对比表
- 习惯 vs 此刻
- 天气形容词 vs 动词-ing

## 写作
- 天气 + 活动（两时态）
