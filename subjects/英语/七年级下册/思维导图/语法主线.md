# 英语七下 语法主线地图

## 一、疑问与描述（Unit 1–2）
### 特殊疑问句进阶
- what/which/who/where/when/why/how
- how many / how much / how often
- 疑问词 + 一般疑问语序
### 形容词
- 定语：a cute panda
- 表语：The tiger is dangerous
### 祈使句
- Be quiet / Don't run
- Please + 动词原形
### 情态动词 have to / must
- have to / has to（不得不）
- must / mustn't（必须/禁止）
- don't have to（不必）≠ mustn't

## 二、代词与饮食（Unit 3–4）
### 物主代词进阶
- 形容词性：my, your, his, her...
- 名词性：mine, yours, his, hers...
### 频度副词
- always → never
- be 后 / 实义动词前
- How often...?
### 选择疑问句
- A or B? — 直接选，不用 Yes
- would like to...
### 可数不可数进阶
- some/any, many/much
- a bowl/glass/piece of...

## 三、进行时（Unit 5–6）
### 现在进行时
- am/is/are + v-ing
- 标志：now, Look!, at the moment
- -ing 变化规则
### 时态辨析
- 习惯/事实 → 一般现在时
- 此刻/暂时 → 现在进行时
- 状态动词不用进行时

## 四、过去时（Unit 7–8）
### 一般过去时（一）
- 规则动词 +ed
- did/didn't + 原形
- 标志：yesterday, last..., ago
### 一般过去时（二）
- was/were
- 不规则动词：go→went, see→saw...
- 讲故事：Once upon a time...
