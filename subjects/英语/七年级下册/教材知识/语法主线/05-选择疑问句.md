# 选择疑问句（Alternative Questions）

## 概念 / 定义
选择疑问句提供**两个或多个选项**，用 **or** 连接，让对方"二选一/多选一"。Unit 4 **Eat Well** 学点餐、选食物时大量出现：Do you want tea **or** coffee? 回答时**不能只说 Yes/No**，要说出选哪一个。

## 公式 · 规则 · 结论（要能直接背）
- **结构**：一般疑问句 + **or** + 选项？ → Do you like apples **or** oranges?
- **回答**：直接说选项，不用 Yes/No。→ — Tea or coffee? — **Coffee, please.**
- **or 连接的成分要平行**：词性相同、结构相同。
  - 名词 or 名词：milk **or** juice
  - 动词 or 动词：swim **or** run
- **would like**（想要，礼貌）：
  - Would you like **some** tea? — Yes, please. / No, thanks.
  - What would you like **to eat**? — I'd like **a bowl of noodles**.
- **与一般疑问句区别**：一般疑问句 Yes/No 即可；选择疑问句必须**点名选项**。

## 典型例题
> 例：— Do you want rice or noodles? — ___  （选面条）
>
> 解：— **Noodles, please.** / **I'd like noodles.**（不说 Yes）

## ⚠️ 易错点 · 陷阱
- ✗ — Tea or coffee? — **Yes**, coffee. → ✓ **Coffee**, please.（别用 Yes 开头）
- ✗ Do you like **apples or orange**? → ✓ ... **oranges**?（or 两边形式平行）
- ✗ Would you like **eat** something? → ✓ Would you like **to eat**...?
- ✗ I'd like **a** rice. → ✓ I'd like **some** rice.（rice 不可数）

## 变式 · 拓展 · 中考链接
- 听力常考选择疑问句，听清 or 两边的选项。
- 写作：模拟餐厅对话，用 would like + 选择疑问句。

## 一问一答（AI 复习时优先用这些来问她）
- Q: 选择疑问句能只答 Yes 吗？ ｜ A: 不能，要说选哪个。
- Q: "你想要茶还是咖啡"怎么说？ ｜ A: Would you like tea or coffee?
- Q: would like 后面加 to 吗？ ｜ A: 加动词时用 to：would like to eat。
