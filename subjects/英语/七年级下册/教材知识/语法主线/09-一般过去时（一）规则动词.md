# 一般过去时（一）规则动词

## 概念 / 定义
一般过去时表示**过去某个时间已经发生的动作或状态**。七下 Unit 7 **A Day to Remember** 开始学"昨天/上周末做了什么"。规则动词过去式一般在词尾加 **-ed**，但读音和拼写有讲究。

## 公式 · 规则 · 结论（要能直接背）
- **结构**：主语 + **动词过去式** + 其他 → I **visited** the zoo yesterday.
- **规则变化**：
  1. 一般 **+ed**：play → played, want → wanted
  2. 以 e 结尾 **+d**：live → lived, like → liked
  3. 辅音+y → 变 y 为 i **+ed**：study → studied, carry → carried
  4. 重读闭音节结尾 → **双写 +ed**：stop → stopped, plan → planned
- **-ed 读音**（中考常考）：
  - /t/：help**ed**, watch**ed**（清辅音后）
  - /d/：play**ed**, clean**ed**（浊辅音或元音后）
  - /ɪd/：want**ed**, visit**ed**（t/d 后）
- **否定/疑问用 did**：
  - 否定：didn't + **动词原形** → I **didn't go**...
  - 疑问：**Did** + 主语 + **动词原形**？ → **Did** you **visit**...?

## 典型例题
> 例：① He ___ (study) English last night. ② ___ you ___ (enjoy) the trip?
>
> 解：
> ① **studied**（辅音+y 变 i +ed）
> ② **Did** you **enjoy**...?（Did 后原形）

## ⚠️ 易错点 · 陷阱
- ✗ I **didn't went**. → ✓ I **didn't go**.（didn't 后原形）
- ✗ **Did** you **visited**? → ✓ **Did** you **visit**?
- ✗ He **studyed**. → ✓ He **studied**.
- ✗ yesterday I **go**. → ✓ yesterday I **went**.（yesterday 提示过去时）

## 变式 · 拓展 · 中考链接
- Unit 7 写作：My Last Weekend — 用一般过去时叙述。
- 八年级会继续深化过去时态，七下先打牢规则变化 + did。
