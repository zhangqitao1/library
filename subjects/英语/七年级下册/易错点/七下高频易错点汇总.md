# 英语七下 · 高频易错点汇总

> 七下语法升级：进行时、过去时、情态动词辨析。复习前先扫一遍，重点攻反复错的。

## 一、特殊疑问句
- ✗ **What** you like? → ✓ **What do you** like?
- ✗ How **many** water → ✓ How **much** water / How many **bottles of** water
- ✗ How **long** do you exercise?（问频率）→ ✓ How **often**...?
- ✗ Why **you** like cats? → ✓ Why **do you** like cats?

## 二、形容词
- ✗ She is **friend**. → ✓ She is **friendly**.
- ✗ very **much** cute → ✓ very **cute**
- ✗ The tiger is **danger**. → ✓ ... **dangerous**

## 三、祈使句与情态动词
- ✗ **Not** run! → ✓ **Don't** run!
- ✗ You **mustn't** wear a hat.（不必戴）→ ✓ You **don't have to** wear a hat.
- ✗ She **musts** go. → ✓ She **must** go.
- ✗ You **mustn't** to go. → ✓ You **mustn't** go.
- ✗ She **have to** ... → ✓ She **has to** ...

## 四、物主代词与频度副词
- ✗ This is **my**. → ✓ This is **mine**.
- ✗ **Mine** book → ✓ **My** book
- ✗ I play **often** basketball. → ✓ I **often** play basketball.
- ✗ I play **one time** a week → ✓ **once** a week

## 五、选择疑问句
- ✗ — Tea or coffee? — **Yes**, tea. → ✓ **Tea**, please.
- ✗ Would you like **eat**? → ✓ Would you like **to eat**?
- ✗ Do you like apples **or orange**? → ✓ ... **oranges**?

## 六、可数不可数（饮食）
- ✗ two **rices** → ✓ two **bowls of** rice
- ✗ a **bread** → ✓ a **piece of** bread / some bread
- ✗ I'd like **a** water → ✓ **some** water / **a glass of** water
- ✗ How **many** milk → ✓ How **much** milk

## 七、现在进行时
- ✗ He **is play** football. → ✓ He **is playing** football.
- ✗ She is **runing**. → ✓ She is **running**.
- ✗ What **do** you doing? → ✓ What **are** you **doing**?
- ✗ They are **studying** every day. → ✓ They **study** every day.（every day→一般现在时）

## 八、一般现在时 vs 现在进行时
- ✗ I **am knowing** the answer. → ✓ I **know** the answer.
- ✗ He **is playing** basketball every day. → ✓ He **plays**...
- ✗ The sun **is going** east. → ✓ The sun **goes** east.
- ✗ It **sun** today. → ✓ It **is sunny** / It **is shining**.

## 九、一般过去时（规则）
- ✗ I **didn't went**. → ✓ I **didn't go**.
- ✗ **Did** you **visited**? → ✓ **Did** you **visit**?
- ✗ He **studyed**. → ✓ He **studied**.
- ✗ yesterday I **go**. → ✓ yesterday I **went**.

## 十、一般过去时（不规则 & be）
- ✗ She **goed** to school. → ✓ She **went**...
- ✗ They **was** happy. → ✓ They **were** happy.
- ✗ I **wasn't** go there. → ✓ I **didn't go** there.
- ✗ Once upon a time, there **is**... → ✓ there **was**...

## 十一、天气与话题表达
- ✗ The weather **are** nice. → ✓ The weather **is** nice.
- ✗ How's the **climate**?（日常）→ ✓ How's the **weather**?
- ✗ **go** swimming → ✓ **go** swimming（✓）vs **play** swimming（✗）

## 通用方法
- **先找标志词定 tense**：now/Look!→进行时；yesterday/last...→过去时；every day/often→一般现在时。
- **助动词后动词原形**：do/does/did/can/must/have to 后面都用原形。
- **选择疑问句别答 Yes**，直接说选项。
