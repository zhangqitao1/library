# 英语 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**英语专属打法**。
> 英语 = 积累 + 运用型，复习核心是"**词汇过关 + 语法会用 + 多读多写**"。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。

## 二、考什么（六大板块）
1. **词汇**：拼写、词义、词性、常见搭配。
2. **语法**：本学期时态（一般现在时/现在进行时等）、句型结构。
3. **句型 / 话题表达**：每单元核心句型与话题必背句。
4. **阅读理解**：抓主旨、猜词、找细节。
5. **完形填空**：靠语境，不是单看一个空。
6. **写作 + 听力**：范文句型、听关键词。

## 二.5、🔴 命题新方向落点（2026 · 济南中考新政，必读）

> 2026 济南改"综合评价招生"，英语命题随之转向：**少考"认不认得语法"，多考"读不读得懂、用不用得上"**。下面把四个方向翻译成英语可操作的练法，每条都给"怎么练 + 给小雨怎么说"。

1. **减记忆、增高阶（少纯语法辨识，多语篇理解与应用）**
   - 怕的是"脱离句子单背规则"。练法：把每个语法点放进**一段话/一段对话**里考，问"为什么这里用进行时""这两句时态为什么不同"，而不是"下列哪项是进行时"。
   - 多问高阶问句：**Why / How do you know / What can we learn**（推断、概括、应用），少问 What is（纯识记）。
   - 给小雨说："别急着选，先告诉我这段在讲什么，再说为什么。"

2. **长情境先提取（真实语篇/对话/应用文，先读关键信息再答）**
   - 题干会给**真实长材料**：通知/海报/菜单/天气预报/日记/小故事/英文对话。练"**先扫读抓关键信息**（谁、何时、何地、数字、转折词 but/however）再答题"。
   - 训练**信息提取三步**：① 先看题目问什么（带着问题读）② 在材料里**画出对应关键词/数字**（回原文定位）③ 比对选项，排除张冠李戴。
   - 给小雨说："题目问什么，你就去文章里找那一句，用笔圈出来——答案藏在原文里，不靠猜。"

3. **读写结合与信息整合（完形/阅读重逻辑推断；书面表达情境化任务）**
   - **完形**：不是看一个空，要**顺着上下文逻辑**（时间线、因果、转折）选词；先把全文读一遍懂大意再回填。
   - **阅读**：除找细节，练**推断题**（文中没明说，要由信息推出）和**整合题**（综合几句话得结论）。
   - **写作**变"情境任务"：给一个真实情境（如给笔友写信介绍我的健康习惯），要求**整合要点 + 用本单元话题语言**完成，不是默写范文。

4. **重过程规范（写作结构、连贯衔接、要点完整，范文示范）**
   - 写作按**四步过程**练：① **审题**（圈出题目要点、人称、时态——介绍习惯用一般现在时，记上周末用一般过去时）② **列点**（把要点列成中文/英文短语，确保不漏点）③ **连句成段**（用衔接词 first / then / also / but / because / finally 把句子串起来，别一句一句蹦）④ **检查**（时态一致、三单 -s、大小写标点）。
   - 给**范文示范**：让她看一篇结构清楚的范文，标出"开头—主体（分点）—结尾"和衔接词，再仿写。
   - 评价标准对她说清："要点全 + 句子通顺连贯 + 时态人称对 + 卷面干净"，比辞藻华丽更得分。

## 三、英语专属方法（怎么高效）
- **单词间隔重复 + 放进句子记**：别孤立背单词表，在例句/话题里记，隔天滚动复现。
- **语法在"用"中练**：讲完一条语法，立刻造句/做语法填空，而不是背规则。
- **阅读/完形重语境（对接新方向②③）**：先让她说"这段大意"，再做题，治"看懂单词却选错"；练"带着问题回原文画关键词"的信息提取，完形顺逻辑（时间/因果/转折）选。详细解题步骤见 `能力方法/完形与阅读解题策略.md`。
- **写作走"过程"不背范文（对接新方向④）**：按"审题→列点→连句（衔接词）→检查"四步练，范文只用来拆结构、标衔接词后仿写，不死默。
- **输出学习法**：让她当小老师讲清"这里为什么用现在进行时"。

## 四、高频坑（优先攻）
- **时态混用**（现在 vs 进行 vs 过去）。
- **词性用错**、第三人称单数 -s。
- **介词 / 固定搭配**。
- 完形只看空、不读上下文。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按单元过词汇表 + 核心句型 + 语法点（合书默写关键词）。
2. **专项突破**：错题归因，专攻最弱（如时态、完形），同类变式练。
3. **限时模拟**：限时整卷，练阅读+作文时间分配。

## 六、当堂自测形式（交给 `exam-maker`）
单词听写、语法填空、句型转换、阅读理解、完形、小作文。

## 七、闭环沉淀
错题入 `错题本/`（错的单词/语法点单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
