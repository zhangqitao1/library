# 英语 · 完形与阅读 解题策略

> 完形和阅读丢分，多半是"**只看一个空/一句话**，没读懂上下文"。这里是可迁移的解题步骤。
> 用法（家教注意）：先让她做一遍说"为什么选这个"，**先诊断她是没读懂还是不会选**，再用策略带；阅读所有题都要回原文找依据。

## 一、完形填空（重语境，不是单空语法题）

### 步骤
1. **跳过空，先通读全文 1 遍**，抓住：讲了什么事、主人公、整体情绪（开心/难过）。**先懂大意**再做题。
2. **逐空做**，每空问三件事：
   - **词义/逻辑**：上下文需要什么意思？（转折 but / 因果 so / 并列 and）
   - **搭配**：这个词常和谁搭配（动词+介词、固定短语）？
   - **语法**：词性对不对、时态/单复数对不对？
3. **代入复读**：选完把句子读顺，前后逻辑通不通。
4. **回头复查**：读完整篇再扫一遍，前面拿不准的常能被后文"照亮"。

### 高频考点/技巧
- **上下文复现**：答案常在前后文出现过同义/原词 → 找线索词。
- **逻辑关系词**决定转折还是顺承。
- **情感一致**：选项情绪要和全文基调一致。

## 二、阅读理解

### 总策略
- **先读题干，再带问题读文**（细节题尤其）；主旨题先速读全文抓大意。

### 按题型对策
- **主旨大意题**（main idea / best title）：抓**首尾段 + 各段首句**；选概括全文的，不选只覆盖一段的（太窄）或文中没有的（太宽）。
- **细节题**（when/where/who/why）：回原文**定位**那句，对照选项；当心同义改写（原文 happy → 选项 pleased）。
- **猜词题**：看该词**前后句**的解释、举例、对比、标点（破折号/括号常给解释），别用母语直觉乱猜。
- **推理题**（infer / imply / probably）：答案是"由原文能推出"，不是原文原话，也**不能脱离原文瞎想**。
- **指代题**（it / they 指什么）：往**前**找最近的名词代入验证。

### 排除法
- 三类错误选项：**与原文相反**、**原文没提**、**以偏概全/太绝对**（all/only/always/never 要警惕）。

## 易丢分提醒
- 完形**只看一个空**不读上下文。
- 阅读**凭印象/常识答**，不回原文定位。
- 细节题被**同义改写**绕过去。
- 主旨题选了"太窄/太宽"的选项。
