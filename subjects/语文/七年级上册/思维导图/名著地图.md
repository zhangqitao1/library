# 七上名著地图

## 朝花夕拾（鲁迅·回忆性散文集·10篇）
### 关键常识
- 原名《旧事重提》
- 是散文集，不是小说集
- 阅读法：精读·略读·浏览
### 重点篇目
- 阿长与《山海经》：阿长买《山海经》
- 五猖会：父亲逼背《鉴略》（批判封建教育）
- 二十四孝图：批判封建孝道
- 父亲的病：庸医害人
- 藤野先生：日本仙台·正直无偏见
- 范爱农：正直爱国·命运凄凉
- 从百草园到三味书屋：童年（课文出处）
### 重点人物
- 长妈妈：善良真诚·令人怀念
- 藤野先生：治学严谨·最敬重的老师
- 范爱农：正直倔强
### 主题
- 批判旧道德旧教育 + 怀念温情往事

## 西游记（吴承恩·明·神魔小说·100回）
### 主线
- 大闹天宫 → 西天取经 → 九九八十一难 → 修成正果
### 主要人物
- 孙悟空：神通广大·机智勇敢·爱憎分明
- 唐僧：慈悲坚定·有时是非不分
- 猪八戒：憨厚·知错能改 / 好吃懒做·贪财
- 沙僧：忠厚老实·任劳任怨
### 经典情节
- 大闹天宫 / 三打白骨精
- 三借芭蕉扇（火焰山·铁扇公主）
- 真假美猴王 / 大战红孩儿
### 主题
- 反抗精神 + 不畏艰险追求理想 + 浪漫主义想象

## 易错提醒
### 朝花夕拾是"散文集"非小说集
### 西游记作者=吴承恩（明），勿写成罗贯中
### 阿长买《山海经》，藤野在日本仙台
