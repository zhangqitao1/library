# 语文七上 · 高频易错点汇总

## 一、默写易错字（默错=直接丢分）
### 古代诗歌四首
- 水何`澹澹`（≠淡淡）、山岛`竦峙`、`星汉灿烂`、东临`碣石`、秋风`萧瑟`
- `子规`啼（≠子归）、`夜郎`西、随`君`直到（课本作"君"）
- `海日生残夜`（≠残月）、`枯藤`老树`昏鸦`、断肠人在`天涯`（≠天崖）
### 课外古诗词
- 碧`霄`（≠宵）、`寂寥`、春`朝`、`戍`轮台（≠戊/戌/戎）、`僵`卧（≠疆）、铁马`冰河`
- `平羌`、`渝州`、`岐王`（≠歧）、`遥怜`、应`傍`、`回乐烽`（≠峰）、`芦管`
- `簇`此城、犹`嫌束`、不`解平`

## 二、文言实词 / 通假 / 活用
### 通假字
- `说`=悦（不亦说乎）、`有`=又（吾十有五）、`亡`=无（身亡所寄 / 亡处亡气）
### 古今异义
- `儿女`（古：子侄辈）、`不`=否（尊君在不）、`去`（古：离开）、`几何`（古：多少）
### 词类活用（《狼》高频）
- `犬坐`于前（名→状，像狗一样）、`洞`其中（名→动，打洞）、`隧`入（名→状，从通道）
### 一词多义
- "敌"：受其`敌`(攻击) / 诱`敌`(敌人)
- "之"：复投`之`(代狼) / 两狼`之`并驱(取独) / 久`之`(凑音节) / 禽兽`之`变诈(的)
- "而"：温故`而`知新(顺承) / 人不知`而`不愠(转折) / 三十`而`立(修饰) / 博学`而`笃志(并列)

## 三、称谓 & 文学常识
- `尊君`=对方父亲；`家君`=自己父亲（《陈太丘与友期》），勿反。
- 《世说新语》编者=**刘义庆**（南朝），笔记小说。
- 《论语》=语录体，记孔子及弟子言行；《诫子书》=诸葛亮**家书**（≠奏章《出师表》）。
- 《狼》出自蒲松龄《聊斋志异》；《穿井得一人》出自《吕氏春秋》，《杞人忧天》出自《列子》。

## 四、名著最易错
- **《朝花夕拾》是散文集，不是小说集**（鲁迅小说集是《呐喊》《彷徨》）。
- 给"我"买《山海经》的是**阿长（长妈妈）**；藤野先生在**日本仙台**。
- 《西游记》作者**吴承恩（明代）**，勿写罗贯中/施耐庵。
- "三借芭蕉扇"地点火焰山、对手铁扇公主（罗刹女）。

## 五、阅读答题套路（别只背术语，要结合内容）
- 概括内容：谁+在什么情况下+做了什么+结果。
- 概括主旨：写了什么 + 表达什么情/理（两部分都要）。
- 句子赏析三步：用了什么手法 → 写出什么 → 表达什么情。
- 环境描写作用：交代背景 / 渲染气氛 / 烘托心情 / 推动情节 / 铺垫。
- 记叙顺序：顺叙(清楚) / 倒叙(设悬念) / 插叙(补充丰富)。

## 六、写作通病
- 流水账（没中心）→ 围绕一个中心选一件事详写。
- 只叙述无描写 → 至少一处动作/语言/神态细节特写。
- 不点题 → 首尾呼应、结尾用议论抒情点明中心。
