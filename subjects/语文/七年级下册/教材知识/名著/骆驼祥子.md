# 名著：《骆驼祥子》

## 概念 / 定义
**老舍**代表作，现代长篇小说。讲人力车夫**祥子"三起三落"**买车梦碎，最终堕落的故事——看清**旧社会怎样把一个好人逼坏**。

## 公式 · 规则 · 结论（要能直接背）

### 基本常识
- 作者：**老舍**（原名舒庆春），现代著名作家，"人民艺术家"。
- 背景：20 世纪 20 年代**北平**（北京）。
- 阅读方法（2024 课本）：**圈点、批注、做笔记**（重点导读）。

### "三起三落"（必背情节线）
| | 起（买车希望） | 落（梦碎） |
|---|---|---|
| **一起一落** | 苦干三年买第一辆车 | 被乱兵抢走 |
| **二起二落** | 卖骆驼攒钱，准备再买 | 被孙侦探敲诈洗劫 |
| **三起三落** | 虎妞买车给祥子 | 虎妞难产卖车办丧事 |

### 主要人物
- **祥子**：老实、健壮、要强、勤劳 → 三起三落后 → 懒惰、狡猾、麻木、堕落。
- **虎妞**：车厂主刘四爷之女，泼辣、精明、自私；既是受害者也是帮凶。
- **刘四爷**：人和车厂老板，残忍自私。
- **小福子**：善良弱女子，祥子最后的精神寄托；上吊自杀成为祥子最后打击。
- **曹先生**：进步知识分子，待祥子较好（相对"善人"）。

### 主题
- 揭露**旧中国黑暗社会**对底层劳动者的摧残。
- 祥子悲剧=**个人奋斗在旧社会必然失败**的典型。

## 典型例题（名著常考）
> 例：祥子"三起三落"分别指什么？这说明了什么？
>
> 解：见上表。说明在**人剥削人、人压迫人**的旧社会，底层劳动者即使勤劳要强，也无法掌握命运，最终精神毁灭。

## ⚠️ 易错点 · 陷阱
- 祥子**不是**一开始就坏——要答**变化过程**（前后对比）。
- 虎妞**不是**纯反派，性格复杂（对祥子有真感情，但手段自私）。
- 书名《骆驼祥子》因**第一次落**后卖骆驼得名。
- 别和《钢铁是怎样炼成的》混淆：保尔是**革命成长**，祥子是**旧社会堕落**。
