# 语文 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**语文专属打法**。
> 语文 = 长期积累型，复习核心是"梳理基础 + 提升运用"，**不是临考刷题**。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**，告诉她"今天重点攻 X、Y，其余快速过"。

## 二、考什么（六大板块）
1. **字词积累**：课本"读读写写"、多音字、形近字、同音字。
2. **古诗文背诵默写**：要求背的古诗词/文言文——会背 + **会默 + 懂意思情感**。
3. **文言文**：重点实词、虚词、句子翻译、作者朝代。
4. **现代文阅读**：记叙文/说明文方法，概括中心、赏析句子、理解词义。
5. **写作**：素材积累 + 结构（总分总/顺序）+ 开头结尾呼应。
6. **文学常识 / 名著**：作者、朝代、代表作、体裁。

## 三、语文专属方法（怎么高效）
- **古诗文滚动默写**：每天默 1~2 篇旧的，间隔重复，重点盯**易错字**（"会背≠会写"）。
- **理解性默写**：不是给上句填下句，而是按"情景/情感"提示默——平时就这么练。
- **阅读题练"套路 + 理解"**：详细答题框架见 `能力方法/现代文阅读答题框架.md`（记叙文/说明文/议论文各题型骨架）；作文结构见 `能力方法/作文结构与升格.md`。先问"这句好在哪"再给框架，不许背模板套。
- **输出学习法**：一篇课文讲完，让她用自己的话复述中心思想/作者怎么写的。
- **作文素材本**：把生活经历、好句子随手记，复习时翻。

## 三·补、命题新方向落点（2026·济南新政）

> 2026 济南综合评价招生重"理解+迁移"，语文命题从"考背诵"转向"考运用"。下面把四个新方向翻译成**语文能直接练的动作**——复习时按这四条调整，别再只刷默写填空。

1. **减记忆、增高阶（少"是什么"，多"为什么/比较/怎么用"）**
   - 默写仍要会，但练法升级：从"给上句填下句"改成**情境式/比较式**（如"同样写愁，李煜和……怎么不同"）。
   - 每讲一篇课文，逼她答三类问题：**为什么**这样写（作者意图）、和谁**比较**（同主题/同手法的篇目对照）、这手法还能**用在哪**（迁移到课外文段/作文）。
   - 文学常识别死背，改成"放进对比表里记"（朝代/体裁/手法成网，见易错点表）。

2. **长情境先提取（真实长材料，先读信息再解题）**
   - 平时练阅读先做一步**"信息提取"**：给一段长材料，先让她圈出"谁、做了什么、关键数据/观点"，再答题——治"脱离原文空答"。
   - 多练**非连续文本**（图表文字化 + 多段材料）：先比对几则材料讲了什么，再整合作答。
   - 古诗文也套长情境：把名句放进一段真实活动/本土文化的语段里考默写与运用。

3. **跨学科 + 探究（语文嵌历史地理、主题活动、信息整合、课内外对比）**
   - **现代文/古诗文嵌背景**：读《木兰诗》联系北朝、读《活板》联系印刷术史、咏物诗联系作者处境——用"背景"解"文本"。
   - **文言文课内外对比**：课内一篇（如《陋室铭》）+ 课外同主题一篇，比异同、迁移实词——这是新卷的固定题型。
   - **主题活动情境任务**：拟宣传语、设计活动、写发言稿、做调查报告——把语文用进真实场景。
   - **非连续文本信息整合**：科技/航天/本土文化等跨学科材料，练"读图表 + 整合多源信息"。

4. **重过程规范（赏析/简答术语规范、答题有步骤）**
   - 赏析题练**三步骨架**：①指出手法/炼字 → ②结合诗句/文本具体分析 → ③点出表达效果与情感。缺一步就扣分。
   - 简答题练**踩点作答**：先看分值定要点数，分点写、用术语（如"细节描写""间接抒情""借景抒情"），不空贴"生动形象"。
   - 复习时**让她照解析的步骤回写一遍**，对照"标准步骤"找差距——名校就在"会不会规范表达"上拉分。

> 一句话：默写照背，但**重头戏挪到"读长材料→提取信息→比较迁移→分步规范作答"**。下面三阶段节奏里，第三阶段的限时模拟要用"新命题方向版"卷子练这套能力。

## 四、高频坑（优先攻）
- 默写**写错字**（同音/形近）—— 默完必须逐字核对。
- 文言**一词多义**、古今异义。
- 阅读题**脱离原文**空答、概括不全。
- 作文偏题、结构散。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按目录过字词/古诗文/课文重点，**合上书复述**，这阶段先不刷题。
2. **专项突破**：错题按"概念不清/审题失误/笔误"归因，专攻最弱板块（如文言翻译）。
3. **限时模拟**：限时做一套综合卷，练阅读+作文的时间分配。

## 六、当堂自测形式（交给 `exam-maker`）
听写默写卷、古诗文理解性默写、文言实词辨析、阅读理解小题、作文提纲训练。

## 七、闭环沉淀
错题入 `错题本/`（默错的字单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
