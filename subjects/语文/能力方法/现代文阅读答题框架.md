# 现代文阅读 · 答题框架

> 阅读题丢分，多半不是读不懂，而是"**答不到点上**"。这里是可迁移的答题套路。
> 用法（家教注意）：**先让她自己答，再对照框架补**——框架是脚手架，不是让她背模板套话。所有题都先问"你从原文哪里看出来的"，**答案必须回原文找依据**。

## 通用第一步：读文 + 标记
1. 先通读，圈出：标题、首尾段、每段中心句、反复出现的词、表情感/转折的词（"但""然而""终于"）。
2. 看题再回读：带着问题精读相关段落，**答题区间先定位到原文**。

## 一、记叙文（写人记事）

### 高频题型 → 答题骨架
- **概括内容/情节**：`谁 + 在什么情况下 + 做了什么 + 结果`（多件事就分条）。
- **概括中心/主旨**：`本文通过……（事），表达/赞美/批判了……（情感/道理）`。
- **赏析句子**：`手法 + 写出了什么内容 + 表达了什么情感`。
  - 手法找：修辞（比喻/拟人/排比…）、描写（动作/语言/神态/心理/环境）、感官、对比。
- **某词能否删去/换**：`不能 + 该词意思 + 在句中的作用 + 删去后变得怎样`。
- **句段作用**：分**内容**（写了什么、表达什么情感）+ **结构**（开头:总领/设悬念/铺垫；中间:承上启下；结尾:总结/点题/呼应开头）。
- **标题含义/作用**：表层意 + 深层意（象征/双关）+ 是否线索 + 是否点明主旨/吸引读者。
- **人物形象**：`从……（具体事/描写）可以看出，他是一个……的人`（先举证据再下结论）。

### 答题铁律
- 一问一句概括 → 一句不够就分条（①②③）。
- 情感类必落到"具体情感词"，别只写"很感动"。

## 二、说明文（讲清一个事物/事理）

- **说明对象 + 特征**：对象是什么、有哪些特点（找各段中心句）。
- **说明方法 + 作用**：`举例子/列数字/作比较/打比方/分类别 + 具体地/准确地/生动地说明了……`。
- **说明顺序**：时间 / 空间 / 逻辑（由总到分、由因到果、由主到次）。
- **语言准确性**（"大约""可能""之一"能否删）：`不能 + 该词表示……（限制/估计）+ 删去后与事实不符 + 体现说明文语言的准确严谨`。

---

## 三、议论文（讲道理、说服人）

- **找中心论点**：常在标题/开头/结尾；是一个完整的肯定句（不是短语、不是问句）。
- **论证方法 + 作用**：`举例论证/道理论证/对比论证/比喻论证 + 有力地/生动地论证了……（分论点）`。
- **论证思路**：`首先提出……，接着用……论证，最后得出/强调……`。
- **某段能否删/调换顺序**：看它和论点、和前后段的逻辑关系。

---

## 易丢分提醒
- **脱离原文空答**——任何概括/赏析都要有原文依据。
- 概括**不全**（多件事只答一件）。
- 赏析只写"用了比喻"却不说写出什么、表达什么。
- 主旨题只答"事"不答"情/理"。
