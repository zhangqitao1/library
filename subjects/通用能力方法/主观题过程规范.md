# 主观题过程规范（拿"过程分"）

> 治什么病：答案"大意对了却扣分"。名校/中考主观题给的是**过程分**——分点清楚、术语准确、有依据，才拿得全。这套规范全科通用，落到笔头。

## 通用三件事

1. **分点作答**：用 ①②③ 或"第一/第二"分开写，一点一个意思，别糊成一段。
2. **术语准确**：用学科词，不用大白话（地理说"地势西高东低"不说"西边高"；生物说"蒸腾作用"不说"出水"）。
3. **有依据**：观点后面跟理由/材料/公式，不能只甩结论。

## 各科"答题结构"模板

- **历史材料题** → **论从史出**：先"由材料……可知"（摆依据）→ 再下结论 → 必要时联系所学补充。
- **道法辨析题** → **观点 + 理由 + 结论**：先判断对不对 → 讲为什么（条文/道理）→ 收一句（对青少年的启示）。
- **地理综合题** → **先因后果/先现象后原因**：用"因为……所以……"，按"位置→地形→气候→人文"顺序展开。
- **生物探究题** → **变量 + 现象 + 结论 + 依据**：控制了什么变量、看到什么现象、得出什么结论、为什么。
- **数学解答题** → **写步骤**：设→列（式/方程）→解→答；几何写"因为…所以…（依据定理）"，化简到底、单位带上。
- **语文赏析题** → **手法 → 分析 → 效果/情感**三步，扣住原文词句。

## 引导式带练

- 先让她按自己的方式答，再对照模板问："你这答案**分点**了吗？""**依据**在哪？""有没有用**学科词**？"
- 让她当小老师说："这类题答题分哪几步？"——能说清才算会。

## 自查（交卷前）

- [ ] 分点了？（看得出 ①②③）
- [ ] 每点有依据/理由，不是只有结论？
- [ ] 用了学科术语？
- [ ] 按本科结构模板走了？
