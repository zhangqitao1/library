# 道法七上 · 高频易错点汇总

> 道法的"错"多在**易混观点**和**主观题不踩点**。先记清易混点，再练答题套路。

## 一、易混观点（最容易张冠李戴）

### 第一单元 少年有梦
- **认识自己的两条途径**：自我评价 ≠ 他人评价，问"途径"两个都要写。
- 对他人评价：**客观冷静**，既不盲从也不忽视（不是"全照别人改"也不是"全不理"）。
- **接纳自己 ≠ 放任缺点**：接纳全部的同时仍要扬长避短、不断完善。
- 少年梦想"两个连接"：与**个人人生目标**相连 + 与**时代/中国梦**相连，缺一不可。
- **努力 = 梦想与现实之间的桥梁**（关键比喻，必背）。

### 第二单元 成长的时空
- 孝亲敬长：既是**传统美德**，又是**法定义务**（两层身份都要答）。
- "家"的内涵答**身心港湾、情感归属**，别只答"住的地方"。
- **教学相长是双向的**：学生也能促进老师，不止老师教学生。
- **友谊讲原则**："哥们义气"≠真友谊（最高频）；友谊是**平等双向**的；友谊会变化是正常的。
- 网络交友：**慎重 + 理性 + 自我保护**，不是"一律拒绝"也不是"完全信任"。
- 个人与集体：**相互依存、利益本质一致**；以集体为重 ≠ 牺牲一切，集体也护个人合理利益。

### 第三单元 珍爱我们的生命
- 生命四特点要记全：**独特 / 不可逆 / 短暂 / 接续**。
- 敬畏生命 = 珍爱**自己** + 关怀**他人**，不能只答一头。
- "增强安全意识"（思想观念）≠ "提高防护能力"（方法技能），分清两层。
- 遇险首要原则：**先保护好自己**；见义"智"为，量力而行、善借外力。
- 健康 = **身体健康 + 心理健康**，别只写一边；调节情绪要写**具体方法**。

### 第四单元 追求美好人生
- 正确人生目标 = **价值导向正确 + 符合实际 + 个人与国家社会需要结合**。
- 顺境：**不骄傲、继续努力**；逆境：**不气馁、迎难而上**（要对应答）。
- "逆境出人才"不绝对——关键在**面对逆境的态度**。
- 人生价值核心 = **对社会和他人的贡献**，不是名利；**劳动是创造价值的源泉**；平凡也能伟大。

## 二、主观题答题套路（道法拿分关键）

### 通用三段式："是什么—为什么—怎么做"
- **是什么**：先判断材料中的现象/行为对不对、属于什么问题。
- **为什么**：用课本观点说明原因、意义或危害（连结论）。
- **怎么做**：给出具体、可操作的做法（分点、落到行动）。

### 常见设问 → 对应套路
| 设问类型 | 答题要点 |
|---|---|
| "谈谈你的认识/看法" | 是什么（判断）+ 为什么（道理）+ 怎么做 |
| "……的意义/重要性" | 对个人 + 对他人 + 对社会/国家，分层答 |
| "你打算怎么做/请提建议" | 分点写**具体行动**，结合自身实际 |
| "为什么这么说" | 摆课本结论 + 联系材料 |
| "评析某人做法" | 先表态对错 → 讲道理 → 正确做法 |

### 踩分小技巧
- **分点作答**，序号清楚（①②③），一点一个得分点。
- **观点 + 材料结合**：光抄课本不结合材料会扣分。
- **从个人到国家分层**，结尾升华（如把个人梦融入中国梦）。
- 字数够、卷面整洁，关键词（如"生命至上""教学相长""劳动创造价值"）要写出来。
