# 道法七下 · 高频易错点汇总

> 七下新增**青春心理、传统文化、民法典**三大块。先记易混点，再练"是什么—为什么—怎么做"三段式。

## 一、易混观点（最容易张冠李戴）

### 第一单元 珍惜青春时光
- 青春期变化是**正常**的 → 要"正视接受"，不是"消除变化"。
- 异性交往：**自然大方、把握分寸**，不是"一律禁止"也不是"放任早恋"。
- **朦胧好感 ≠ 早恋**：正常心理 vs 不当行为要分清。
- **管理情绪 ≠ 压抑情绪**：要合理表达、合理宣泄（健康方式），不能伤害他人。
- **情绪 vs 情感**：情绪短暂易变；情感稳定深刻——问"区别"别混。
- 合理宣泄 = 倾诉、运动等；**摔东西、吼人 ≠ 合理宣泄**。

### 第二单元 焕发青春活力
- **自尊 ≠ 虚荣 ≠ 自负 ≠ 自卑**：自尊重内在价值；虚荣重面子；自负盲目自大；自卑妄自菲薄。
- **自信靠实力和努力**，不是空喊"我能行"。
- **自强 ≠ 蛮干 ≠ 拒绝一切帮助**：自强是主要依靠自己，合理求助不算依赖。
- 问"做自强不息的中国人"要**升华到民族传统、时代责任**，不能只答个人层面。
- 自尊、自信、自强**相互联系**，综合题常三者一起考，各答各的内涵。

### 第三单元 传承中华优秀传统文化
- **思想理念 / 人文精神 / 传统美德**三层别混：
  - 思想理念 → 治国理政的价值原则
  - 人文精神 → 做人做事的精神品格
  - 传统美德 → 具体道德规范和行为准则
- **"和而不同" ≠ 无原则一团和气**：和=和谐，不同=尊重差异。
- 传承 ≠ 简单复古 → 要**创造性转化、创新性发展**。
- **"见利思义"** = 义利兼顾、以义为先，不是"不要任何利益"。
- ⚠️待核对：第三单元各课具体列举条目，开学对照课本最终确认。

### 第四单元 生活在法治社会
- **法律 vs 道德**：法律靠国家强制力；道德靠社会舆论和内心信念——相辅相成，别混。
- 法律作用要答**规范 + 保护**，不只答"惩罚"。
- **宪法是根本法**，地位最高。
- **人身权 ≠ 财产权**：肖像权、隐私权属人身权；盗窃侵犯财产权。
- **一般违法 ≠ 犯罪**：看社会危害程度；犯罪三个特征**缺一不可**。
- **拘役 ≠ 拘留**，**管制 ≠ 管理**——刑罚名称别写错。
- 维权走**合法途径**（告知、调解、诉讼），不能"以牙还牙"。
- ⚠️待核对：七年级对刑事责任年龄、主刑附加刑的展开程度，以课本为准。

## 二、主观题答题套路（道法拿分关键）

### 通用三段式："是什么—为什么—怎么做"
- **是什么**：判断现象/行为，点明涉及什么观点/权利/行为性质。
- **为什么**：用课本结论说明意义、危害或原因。
- **怎么做**：分点写**具体、可操作**的行动。

### 七下常见设问 → 对应套路
| 设问类型 | 答题要点 |
|---|---|
| "如何管理情绪" | 具体方法（改变认知、转移注意、合理宣泄……） |
| "评析攀比/虚荣行为" | 是什么（虚荣非自尊）→ 为什么 → 怎么做 |
| "体现了什么思想理念/美德" | 准确名称 + 内涵 + 当代价值/践行 |
| "侵犯了什么权利" | 人身权/财产权分类 + 依法维权途径 |
| "属于什么违法行为" | 判断性质 + 危害 + 法律后果 + 警示 |
| "如何远离违法犯罪" | 防微杜渐 + 学法守法 + 道德修养 + 抵制诱惑 |

### 踩分小技巧
- **分点作答**，关键词要写出来（如"防微杜渐""创造性转化""肖像权"）。
- **观点 + 材料结合**：光抄课本不结合材料会扣分。
- 传统文化题结尾可**升华到文化自信**；法治题可升华到**做法治小公民**。

## 三、与七上的衔接提醒
- 七上"友谊讲原则""敬畏生命""人生价值" → 七下"情绪管理""自尊自强""传统美德"可跨册综合。
- 七上已学"法律保障生活"（上册第四单元前导）→ 七下第四单元**深化民法典与违法犯罪**，别用旧版"走进法治天地"四课结构复习。
