# 道德与法治 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**道法专属打法**。
> 道法 = 概念 + 情境运用型，复习核心是"**懂概念 + 会用材料题答全**"，不是死背条目。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。

## 二、考什么（三大维度）
1. **核心概念 / 观点**：理解每课要传达的道理，不只背关键词。
2. **生活情境应用**：用观点分析一个生活/时事情境（材料分析题为主）。
3. **辨析 / 判断**：判断说法对错并说理由。

## 三、道法专属方法（怎么高效）
- **概念联系实际**：每个观点配一个生活例子，治"会背不会用"。
- **材料题答题结构**：**是什么 → 为什么 → 怎么做**，分点作答，**紧扣材料**不空谈。
- **关键词分点**：答案踩点给分，先提炼关键词再展开。
- **输出学习法**：让她举一个身边例子说明某个观点。
- **结合时政**（命题趋势）：把课本观点和近期新闻/生活事件挂钩。

## 四、高频坑（优先攻）
- **概念混**（相近观点分不清）。
- **材料题答不全面**（只答一两点）、**脱离材料**空喊口号。
- 辨析题只判断、不说理由。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按目录过核心观点，合书用自己话复述。
2. **专项突破**：错题归因，专攻材料分析题的答题结构。
3. **限时模拟**：限时整卷，重点练材料题分点+踩关键词。

## 六、当堂自测形式（交给 `exam-maker`）
概念辨析、判断说理、生活情境材料分析题、观点+实例配对。

## 七、闭环沉淀
错题入 `错题本/`（答不全的材料题、混淆的概念单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
