# 道德与法治 · 复习流程（最佳实践）

> 通用引导式与循证学习法见 `AGENTS.md` 三.5、`.cursor/skills/review-coach/`；本文件是**道法专属打法**。
> 道法 = 概念 + 情境运用型，复习核心是"**懂概念 + 会用材料题答全**"，不是死背条目。

## 一、开练前 · 诊断（必做）
读 `学情/小雨-学情档案.md` + 该册 `00-册地图.md` + `错题本/`，定 **2~3 个今日重点**。

## 二、考什么（三大维度）
1. **核心概念 / 观点**：理解每课要传达的道理，不只背关键词。
2. **生活情境应用**：用观点分析一个生活/时事情境（材料分析题为主）。
3. **辨析 / 判断**：判断说法对错并说理由。

## 三、道法专属方法（怎么高效）
- **概念联系实际**：每个观点配一个生活例子，治"会背不会用"。
- **材料题答题结构**：**是什么 → 为什么 → 怎么做**，分点作答，**紧扣材料**不空谈。
- **关键词分点**：答案踩点给分，先提炼关键词再展开。
- **输出学习法**：让她举一个身边例子说明某个观点。
- **结合时政**（命题趋势）：把课本观点和近期新闻/生活事件挂钩。

## 三.5、命题新方向落点（2026 · 据济南中考新政翻译成道法可操作打法）

> 背景：2026 起济南综合评价招生看生地等级、整体命题"减记忆增高阶、重情境与运用"。道法是等级科、偏基础，但**新方向不是更难，而是更"活"**——少背条文、多用条文。四个落点逐条会做：

1. **减记忆、增高阶——别只问"是什么"，多练"为什么/比较/评价/怎么用"。**
   - 每复习一个观点，立刻追问自己三连："这观点想说啥（是什么）→ 为什么对/重要 → 生活里哪件事能用上"。
   - 易混概念**成对比较**记（自尊 vs 虚荣 vs 自负、情绪 vs 情感、一般违法 vs 犯罪、法律 vs 道德），练"说出异同"而不是各背一段。

2. **长情境先提取——先读懂材料，再下笔。**
   - 材料题养成"**圈关键词**"习惯：先把材料里的人物行为、转折词、时政热点圈出来，想清楚"它考哪个观点"，**答案必须回扣材料原话**，不能甩一段背诵。
   - 练时政/生活真实情境：非遗国潮、博物馆热、民法典、网络保护未成年人、身边的自强榜样——把课本观点往这些场景上套。

3. **综合运用 + 价值引领——往"家国 + 法治 + 担当"上升华，会跨情境迁移。**
   - 答完小问，想一句"**对我们青少年的启示**"：传统文化题升华到**文化自信**，法治题升华到**做尊法学法守法用法的好公民**，成长题升华到**自强担当、报效国家**。
   - 练"一料多问"和"跨单元综合"：一段材料同时能用青春、传统文化、法治的知识答，刻意混着练（提取网格）。

4. **重过程规范——辨析/简答**按固定结构分点，**踩点给分**。
   - **辨析题三步**：①**亮观点**（这个说法是对的/错的/不全面）→ ②**讲理由**（用课本观点说为什么，正反都说）→ ③**下结论/给启示**（应该怎么做）。光判断不说理 = 半分都难拿。
   - **材料/简答**：先写**关键词**再展开，**分点**作答（①②③），每点"观点 + 扣材料"，宁可点多句短，别一大段糊上去。
   - 复习时让小雨把答案**先口头分点说**，再落笔，治"会想不会写规范"。

## 四、高频坑（优先攻）
- **概念混**（相近观点分不清）。
- **材料题答不全面**（只答一两点）、**脱离材料**空喊口号。
- 辨析题只判断、不说理由（违反三.5 第 4 条，最常丢分）。
- **审长材料丢信息**：没圈关键词就下笔，答非所问（违反三.5 第 2 条）。

## 五、期末三阶段节奏（对标历下）
1. **回归课本**：按目录过核心观点，合书用自己话复述。
2. **专项突破**：错题归因，专攻材料分析题的答题结构。
3. **限时模拟**：限时整卷，重点练材料题分点+踩关键词。

## 六、当堂自测形式（交给 `exam-maker`）
概念辨析、判断说理、生活情境材料分析题、观点+实例配对。

## 七、闭环沉淀
错题入 `错题本/`（答不全的材料题、混淆的概念单独记）→ 更新 `00-册地图.md` 掌握度 → `学情/复习日志.md` 追加一行。
